# 数据采集服务设计

## 1. 模块概述

数据采集模块负责从外部数据源采集A股行情数据、财经新闻数据，为其他模块提供数据支撑。

## 2. 技术架构

### 2.1 技术栈

| 组件 | 技术选型 | 说明 |
|------|---------|------|
| 数据源 | AKTools API | A股行情数据 |
| 数据源 | 新浪财经/东方财富 | 财经新闻爬取 |
| HTTP客户端 | Hutool HttpUtil | HTTP请求 |
| 数据存储 | MySQL | 股票列表、历史行情 |
| 定时任务 | Quartz | 定时同步任务 |

## 3. 服务设计

### 3.1 StockDataService 接口

```java
public interface StockDataService {
    
    /**
     * 获取A股全市场股票列表
     * @return 可交易股票列表（排除ST、停牌等）
     */
    List<StockInfo> fetchStockList();
    
    /**
     * 获取个股历史K线数据
     * @param stockCode 股票代码
     * @param startDate 开始日期
     * @param endDate 结束日期
     * @return 历史价格列表
     */
    List<StockPrices> fetchHistoricalData(String stockCode, LocalDate startDate, LocalDate endDate);
    
    /**
     * 获取个股近N天历史数据
     * @param stockCode 股票代码
     * @param days 天数
     * @return 历史价格列表
     */
    List<StockPrices> fetchHistoricalData(String stockCode, int days);
    
    /**
     * 获取实时行情数据
     * @param stockCode 股票代码
     * @return 当前价格
     */
    Double fetchRealTimePrice(String stockCode);
    
    /**
     * 获取财务报表数据
     * @param stockCode 股票代码
     * @return 财务数据JSON
     */
    String fetchFinancialReport(String stockCode);
    
    /**
     * 同步所有股票数据到本地数据库
     */
    void syncAllStocks();
    
    /**
     * 同步指定股票的历史数据
     * @param stockCode 股票代码
     * @param days 同步天数
     */
    void syncStockHistory(String stockCode, int days);
}
```

## 4. 数据模型

### 4.1 StockInfo 股票信息

```java
@Entity
public class StockInfo {
    @Id
    private String stockCode;        // 股票代码 (如: 600519)
    private String stockName;        // 股票名称 (如: 贵州茅台)
    private String market;           // 所属市场 (SH/SZ)
    private String industry;         // 所属行业
    private String listingDate;      // 上市日期
    private Boolean isST;           // 是否ST
    private Boolean isTradable;     // 是否可交易
    private LocalDateTime updatedAt; // 更新时间
}
```

### 4.2 StockPrices 历史行情

```java
@Entity
public class StockPrices {
    @Id
    private String id;               // stockCode + date
    
    private String stockCode;        // 股票代码
    private LocalDate tradeDate;    // 交易日期
    
    private BigDecimal open;        // 开盘价
    private BigDecimal high;        // 最高价
    private BigDecimal low;         // 最低价
    private BigDecimal close;       // 收盘价
    private BigDecimal volume;      // 成交量
    private BigDecimal amount;      // 成交额
    private BigDecimal change;      // 涨跌幅
    private BigDecimal turnover;    // 换手率
    
    private LocalDateTime createdAt;
    private LocalDateTime updatedAt;
}
```

## 5. 定时任务

### 5.1 任务配置

| 任务名称 | 执行时间 | 频率 | 说明 |
|----------|----------|------|------|
| 股票列表同步 | 9:00 | 每日 | 同步全市场股票列表 |
| 实时行情同步 | 9:00-14:55 | 每分钟 | 实时行情更新 |
| 历史数据同步 | 14:30-23:00 | 每日 | 收盘后补充历史数据 |
| 新闻采集 | 9:30-11:30, 13:00-15:00 | 每30分钟 | 盘中财经新闻 |

### 5.2 Cron 表达式

```java
@Scheduled(cron = "0 0 9 * * *")                    // 每日 09:00
@Scheduled(cron = "0 0/1 9-14 * * *")               // 9:00-14:55 每分钟
@Scheduled(cron = "0 30 14 * * *")                  // 每日 14:30
@Scheduled(cron = "0 0/30 9-11,13-15 * * *")       // 9:30-11:30, 13:00-15:00
```

public class StockDataServiceImpl implements StockDataService {

## 8. 错误处理

### 8.1 异常类型

| 异常 | 说明 | 处理策略 |
|------|------|----------|
| DataSourceException | 数据源故障 | 切换备用源，重试 |
| RateLimitException | 频率限制 | 等待后重试 |
| ParseException | 数据解析失败 | 记录日志，返回null |
| NetworkException | 网络异常 | 重试3次，告警 |
