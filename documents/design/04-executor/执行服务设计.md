# executor 模块 - 执行服务

## 模块说明

executor 模块负责风控检查和交易执行，是系统的交易执行层。

## 包含功能

| 序号 | 功能 | 说明 |
|------|------|------|
| 1 | 风控检查 | 止损、仓位、熔断检查 |
| 2 | 交易执行 | 买入、卖出、撤单 |
| 3 | 持仓管理 | 持仓查询、盈亏统计 |
| 4 | 账户管理 | 账户信息、资金管理 |

## 核心类

| 类名 | 说明 |
|------|------|
| RiskController | 风控检查器 |
| StopLossChecker | 止损检查 |
| PositionChecker | 仓位检查 |
| TradeExecutor | 交易执行器 |
| OrderManager | 委托管理 |
| PositionManager | 持仓管理 |
| BrokerAdapter | 券商适配器 |

## 风控规则

| 规则 | 阈值 | 说明 |
|------|------|------|
| 日亏损 | 3% | 超过禁止买入 |
| 月亏损 | 10% | 超过禁止买入 |
| 单股仓位 | 30% | 超过禁止买入 |
| 总仓位 | 80% | 超过禁止买入 |

## 定时任务

| 任务 | Cron | 说明 |
|------|------|------|
| morningTrade | 0 35 9 * * 1-5 | 早盘交易 |
| afternoonTrade | 0 50 14 * * 1-5 | 尾盘交易 |

## 对外API

- `POST /api/trade/buy` - 买入下单
- `POST /api/trade/sell` - 卖出下单
- `POST /api/trade/cancel/{orderId}` - 撤单
- `GET /api/trade/position` - 持仓查询
- `GET /api/trade/account` - 账户信息
- `POST /api/trade/risk/check` - 风控检查
