# strategy 模块 - 策略服务

## 模块说明

strategy 模块负责选股策略和交易决策，是系统的核心业务逻辑层。

## 包含功能

| 序号 | 功能 | 说明 |
|------|------|------|
| 1 | 综合选股 | 双因子(LSTM+情感)选股 |
| 2 | 决策引擎 | 交易信号生成 |
| 3 | 策略管理 | 策略配置和执行 |

## 核心类

| 类名 | 说明 |
|------|------|
| StockSelector | 选股器 |
| FactorCalculator | 因子计算 |
| RankingService | 排名服务 |
| DecisionEngine | 决策引擎 |
| SignalGenerator | 信号生成 |
| StrategyManager | 策略管理 |

## 选股算法

```
综合得分 = LSTM得分 × 0.6 + 情感得分 × 0.4
```

## 定时任务

| 任务 | Cron | 说明 |
|------|------|------|
| runStockSelection | 0 0 17 * * 1-5 | 综合选股 |

## 对外API

- `GET /api/strategy/select` - 获取推荐股票
- `GET /api/strategy/select/rank` - 股票排名
- `POST /api/strategy/decision` - 生成决策
- `GET /api/strategy/signal` - 获取交易信号
