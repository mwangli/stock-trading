# 综合选股算法设计

## 1. 模块概述

综合选股模块负责整合情感分析、LSTM预测、Dexter分析三个因子，计算股票综合评分，选出最优股票。

## 2. 算法设计

### 2.1 评分公式

```
综合得分 = LSTM预测因子 × 0.4 + 情感分析因子 × 0.3 + Dexter基本面因子 × 0.3
```

### 2.2 因子计算

| 因子 | 权重 | 计算方式 | 数据来源 |
|------|------|----------|----------|
| LSTM预测 | 0.4 | 基于预测涨跌幅转换 | LSTM预测服务 |
| 情感分析 | 0.3 | 基于情感得分转换 | 情感分析服务 |
| Dexter基本面 | 0.3 | 基于综合评分 | Dexter分析服务 |

### 2.3 得分转换

各因子得分统一转换为 0-100 分：

```
LSTM因子得分 = (预测涨跌幅 + 10) / 20 × 100  # 范围: 0-100
情感因子得分 = (情感得分 + 1) / 2 × 100      # 范围: 0-100
Dexter因子得分 = 综合评分                     # 范围: 0-100
```

## 3. 选股流程

```
输入: 候选股票列表 (默认取热度前100只)
           │
           ▼
┌─────────────────────────────────────────────────────────────┐
│  Step 1: 并行获取三因子数据                                   │
│  ┌──────────────┐ ┌──────────────┐ ┌──────────────┐       │
│  │ LSTM预测     │ │ 情感分析     │ │ Dexter分析   │       │
│  │ 并行调用     │ │ 并行调用     │ │ 并行调用     │       │
│  └──────────────┘ └──────────────┘ └──────────────┘       │
└─────────────────────────────────────────────────────────────┘
           │
           ▼
┌─────────────────────────────────────────────────────────────┐
│  Step 2: 计算综合得分                                        │
│  Score = LSTM×0.4 + Sentiment×0.3 + Dexter×0.3            │
└─────────────────────────────────────────────────────────────┘
           │
           ▼
┌─────────────────────────────────────────────────────────────┐
│  Step 3: 排序筛选                                           │
│  - 按综合得分降序排列                                        │
│  - 选出: Top1 (主推) + Top3 (备选)                          │
└─────────────────────────────────────────────────────────────┘
           │
           ▼
┌─────────────────────────────────────────────────────────────┐
│  Step 4: 过滤排除                                            │
│  - 排除ST股票                                               │
│  - 排除涨停/跌停股票                                         │
│  - 排除停牌股票                                             │
└─────────────────────────────────────────────────────────────┘
           │
           ▼
输出: 主推股票(1只) + 备选股票(3只)
```

## 4. 数据结构

### 4.1 选股请求

```java
public class StockSelectionRequest {
    private List<String> stockCodes;      // 候选股票列表
    private int topN = 3;                // 选出数量
    private LocalDate analysisDate;       // 分析日期
}
```

### 4.2 选股结果

```java
public class StockSelectionResult {
    private String stockCode;             // 股票代码
    private String stockName;             // 股票名称
    private BigDecimal overallScore;     // 综合得分 (0-100)
    
    // 各因子得分
    private BigDecimal lstmScore;        // LSTM因子得分
    private BigDecimal sentimentScore;   // 情感因子得分
    private BigDecimal dexterScore;      // Dexter因子得分
    
    // 预测数据
    private BigDecimal predictedChange;  // 预测涨跌幅
    private String sentimentLabel;       // 情感标签
    private String suggestion;           // 交易建议
    
    private LocalDateTime analyzedAt;   // 分析时间
}
```

### 4.3 选股响应

```java
public class SelectionResponse {
    private List<StockSelectionResult> recommendations;  // 推荐股票列表
    private int totalAnalyzed;            // 分析股票总数
    private LocalDateTime analysisTime;  // 分析耗时
    private LocalDateTime generatedAt;   // 生成时间
}
```

## 5. 因子权重配置

### 5.1 默认配置

```yaml
stock:
  selection:
    weights:
      lstm: 0.4
      sentiment: 0.3
      dexter: 0.3
    top-n: 3
    candidate-limit: 100
```

### 5.2 权重调整策略

| 市场环境 | LSTM | Sentiment | Dexter |
|---------|------|-----------|--------|
| 趋势行情 | 0.5 | 0.2 | 0.3 |
| 震荡行情 | 0.3 | 0.4 | 0.3 |
| 价值投资 | 0.2 | 0.2 | 0.6 |

## 6. 缓存策略

| 数据 | 缓存时间 | 说明 |
|------|----------|------|
| 股票列表 | 1天 | 每日更新 |
| 实时行情 | 1分钟 | 频繁更新 |
| 情感分析 | 1小时 | 新闻时效性 |
| LSTM预测 | 30分钟 | 盘中有效 |
| Dexter分析 | 1小时 | 基本面稳定 |
| 选股结果 | 10分钟 | 盘中有效 |

## 8. 与其他模块集成

### 8.1 依赖模块

| 模块 | 接口 | 用途 |

### 8.2 服务调用

```java
@Service
public class StockSelectionService {
    
    public SelectionResponse selectStocks(StockSelectionRequest request) {
        // 1. 获取候选股票
        List<String> candidates = getCandidates(request.getStockCodes());
        
        // 2. 并行获取三因子
        CompletableFuture<LSTMResult> lstmFuture = lstmService.predictAsync(candidates);
        CompletableFuture<SentimentResult> sentimentFuture = sentimentService.analyzeAsync(candidates);
        CompletableFuture<DexterResult> dexterFuture = dexterService.analyzeAsync(candidates);
        
        // 3. 等待所有结果
        CompletableFuture.allOf(lstmFuture, sentimentFuture, dexterFuture).join();
        
        // 4. 计算综合得分并排序
        List<StockSelectionResult> results = calculateScores(...);
        
        // 5. 过滤和返回
        return filterAndReturn(results, request.getTopN());
    }
}
```
