## 2. 概述

### 2.1 模块定位

综合选股模块是股票交易系统的核心决策模块，负责整合情感分析、LSTM预测、Dexter分析三个因子，计算股票综合评分，选出最优股票供决策引擎使用。

### 2.2 功能摘要

- 整合三因子（LSTM预测、情感分析、Dexter基本面）计算综合得分
- 并行获取各因子数据，提高选股效率
- 按综合得分排序，选出Top N股票
- 过滤不符合条件的股票（ST、停牌、涨跌停）

### 2.3 上下游依赖

| 上游模块 | 接口 | 数据 |
|----------|------|------|
| LSTM预测 | predict() | 预测因子 |
| 情感分析 | analyze() | 情感因子 |
| Dexter分析 | analyze() | 基本面因子 |
| 数据采集 | getStockInfo() | 候选股票列表 |

| 下游模块 | 接口 | 数据 |
|----------|------|------|
| 决策引擎 | selectStocks() | 推荐股票列表 |

### 3.2 功能详情

#### F-001: 三因子整合

**功能描述**: 并行获取LSTM预测因子、情感分析因子、Dexter基本面因子。

**输入**: stockCodes: 候选股票列表

**处理逻辑**:
1. 并行调用LSTM预测服务
2. 并行调用情感分析服务
3. 并行调用Dexter分析服务
4. 等待所有结果返回

**输出**: 各股票的三个因子数据

**业务规则**:
- 使用CompletableFuture并行调用
- 单一服务失败不影响其他服务
- 设置超时时间避免无限等待

#### F-002: 综合评分

**功能描述**: 根据权重计算股票综合得分。

**输入**: 三因子数据

**处理逻辑**:
1. 将各因子转换为0-100分制
2. 按权重计算加权平均
3. 返回综合得分

**输出**: 综合得分 (0-100)

**业务规则**:
- 权重: LSTM 40%, 情感30%, Dexter 30%
- 得分范围: 0-100

**评分公式**:
```
综合得分 = LSTM预测因子 × 0.4 + 情感分析因子 × 0.3 + Dexter基本面因子 × 0.3
```

**得分转换**:
```
LSTM因子得分 = (预测涨跌幅 + 10) / 20 × 100  # 范围: 0-100
情感因子得分 = (情感得分 + 1) / 2 × 100      # 范围: 0-100
Dexter因子得分 = 综合评分                       # 范围: 0-100
```

#### F-003: 排序筛选

**功能描述**: 按综合得分降序排列，选出Top N股票。

**输入**: 股票得分列表

**处理逻辑**:
1. 按综合得分降序排列
2. 选出Top N股票
3. 区分主推和备选

**输出**: 推荐股票列表

**业务规则**:
- Top 1: 主推股票
- Top 3: 备选股票

## 6. 详细设计

### 6.1 服务设计

#### 6.1.1 服务类图

```
StockSelectionService (接口)
    ↑
    └─ StockSelectionServiceImpl (实现)
            │
            ├─ FactorCalculator (因子计算)
            ├─ StockFilter (股票过滤)
            └─ SelectionCache (选股缓存)
```

#### 6.1.2 核心服务接口

```java
public interface StockSelectionService {
    
    /**
     * 选股入口
     * @param request 选股请求
     * @return 选股响应
     */
    SelectionResponse selectStocks(StockSelectionRequest request);
}
```

### 6.2 数据模型设计

#### 6.2.1 选股请求

```java
@Data
public class StockSelectionRequest {
    private List<String> stockCodes;    // 候选股票列表
    private int topN = 3;              // 选出数量
    private LocalDate analysisDate;    // 分析日期
}
```

#### 6.2.2 选股结果

```java
@Data
public class StockSelectionResult {
    private String stockCode;           // 股票代码
    private String stockName;           // 股票名称
    private BigDecimal overallScore;   // 综合得分 (0-100)
    
    // 各因子得分
    private BigDecimal lstmScore;      // LSTM因子得分
    private BigDecimal sentimentScore; // 情感因子得分
    private BigDecimal dexterScore;    // Dexter因子得分
    
    // 预测数据
    private BigDecimal predictedChange; // 预测涨跌幅
    private String sentimentLabel;     // 情感标签
    private String suggestion;         // 交易建议
    
    private LocalDateTime analyzedAt;  // 分析时间
}
```

#### 6.2.3 选股响应

```java
@Data
public class SelectionResponse {
    private List<StockSelectionResult> recommendations;  // 推荐股票列表
    private int totalAnalyzed;         // 分析股票总数
    private long analysisTimeMs;       // 分析耗时(毫秒)
    private LocalDateTime generatedAt; // 生成时间
}
```

## 7. 错误处理

### 7.1 错误码定义

| 错误码 | 错误信息 | 说明 |
|--------|----------|------|
| SEL_001 | 候选股票列表为空 | 无可选股票 |
| SEL_002 | 因子服务调用失败 | 单因子获取失败 |
| SEL_003 | 所有因子获取失败 | 无法计算得分 |

### 7.2 异常处理策略

| 异常类型 | 处理策略 |
|----------|----------|
| 单因子失败 | 使用默认值50分 |
| 网络超时 | 使用默认值50分 |
| 全部失败 | 返回空列表 |

### 7.3 默认因子值

| 因子 | 默认得分 |
|------|----------|
| LSTM | 50 |
| 情感 | 50 |
| Dexter | 50 |
