# 中文情感分析模型下载指南

## 概述

系统已切换为使用**中文 BERT 模型**（哈工大 `chinese-bert-wwm-ext`），需要进行模型下载或手动放置。

## 模型信息

- **模型名称**：哈工大中文 BERT (chinese-bert-wwm-ext)
- **发布机构**：哈工大讯飞联合实验室 (HFL)
- **模型大小**：约 400MB
- **语言**：中文
- **适合场景**：中文文本分类、情感分析、命名实体识别等

## 下载方式

### 方式 1：使用 Git 克隆（推荐）

```bash
# 1. 进入模型目录
cd D:\ai-stock-trading\models\sentiment-analysis

# 2. 安装 Git LFS（如果未安装）
# Windows: 从 https://git-lfs.com 下载安装
# 验证安装：git lfs --version

# 3. 初始化 LFS
git lfs install

# 4. 克隆模型
git clone https://huggingface.co/hfl/chinese-bert-wwm-ext

# 5. 将文件移动到 sentiment-analysis 目录
# （如果在子目录中）
mv chinese-bert-wwm-ext/* .
rmdir chinese-bert-wwm-ext
```

### 方式 2：手动下载

1. **访问 HuggingFace**：
   - 网址：https://huggingface.co/hfl/chinese-bert-wwm-ext

2. **下载必需文件**：
   - `pytorch_model.bin` (约 400MB) - 模型权重
   - `config.json` (约 1KB) - 模型配置
   - `vocab.txt` (约 200KB) - 词表

3. **放置文件**：
   ```
   D:\ai-stock-trading\models\sentiment-analysis\
   ├── pytorch_model.bin
   ├── config.json
   └── vocab.txt
   ```

### 方式 3：使用 Python 脚本下载

如果你有 Python 环境，可以使用以下脚本：

```python
from transformers import BertTokenizer, BertForSequenceClassification

# 下载并保存模型
model = BertForSequenceClassification.from_pretrained("hfl/chinese-bert-wwm-ext", num_labels=3)
tokenizer = BertTokenizer.from_pretrained("hfl/chinese-bert-wwm-ext")

# 保存到指定目录
save_path = "D:/ai-stock-trading/models/sentiment-analysis"
model.save_pretrained(save_path)
tokenizer.save_pretrained(save_path)

print(f"模型已保存到：{save_path}")
```

## 常见问题

### Q1: 下载速度很慢怎么办？

**解决方案**：
- 使用镜像站点：https://hf-mirror.com
- 修改下载命令：
  ```bash
  git clone https://hf-mirror.com/hfl/chinese-bert-wwm-ext
  ```

### Q2: 模型文件太大，有轻量级选择吗？

**是的**，可以选择以下轻量级模型：

1. **RoBERTa-wwm-ext-mini** (约 60MB)
   ```yaml
   pretrained-model: hfl/chinese-roberta-wwm-ext-mini
   ```

2. **MacBERT-small** (约 120MB)
   ```yaml
   pretrained-model: hfl/chinese-macbert-small
   ```

修改 `application.yml` 中的 `pretrained-model` 配置即可。

### Q3: 模型加载失败怎么办？

**检查清单**：
- [ ] 模型文件是否完整下载
- [ ] `pytorch_model.bin` 文件大小是否正确（约 400MB）
- [ ] `config.json` 格式是否正确
- [ ] 是否有足够的内存（建议 2GB+）

**解决方法**：
```bash
# 删除已下载的文件，重新下载
rm -rf models/sentiment-analysis/*
# 重新下载
```

### Q4: 可以使用 GPU 加速吗？

**可以**，如果有 NVIDIA GPU：

1. 确保已安装 CUDA 和 cuDNN
2. DJL 会自动检测并使用 GPU
3. 在日志中查找 `Using GPU` 确认

## 替代模型选择

如果哈工大 BERT 不适合，可以尝试其他中文模型：

| 模型名称 | 大小 | 特点 | 配置值 |
|---------|------|------|--------|
| 哈工大 BERT wwm | 400MB | 综合性能好，推荐 | `hfl/chinese-bert-wwm-ext` |
| 哈工大 RoBERTa | 400MB | 更准确，训练时间长 | `hfl/chinese-roberta-wwm-ext` |
| 哈工大 MacBERT | 400MB | 针对性优化 | `hfl/chinese-macbert-base` |
| Google BERT 中文 | 400MB | 官方版本 | `bert-base-chinese` |
| RoBERTa-mini | 60MB | 轻量级，速度快 | `hfl/chinese-roberta-wwm-ext-mini` |

修改 `application.yml`：
```yaml
models:
  sentiment:
    pretrained-model: hfl/chinese-roberta-wwm-ext-mini  # 更换模型
```
