# intelligence 模块 - 智能服务

## 模块说明

intelligence 模块负责AI训练、推理和模型管理，基于DJL框架实现纯Java深度学习。

## 包含功能

| 序号 | 功能 | 说明 |
|------|------|------|
| 1 | 情感分析 | DJL CNN文本分类 |
| 2 | LSTM预测 | DJL LSTM时序预测 |
| 3 | 模型训练 | 纯Java端到端训练 |
| 4 | 模型管理 | 版本管理、加载、切换 |

## 核心类

| 类名 | 说明 |
|------|------|
| StockLSTM | LSTM网络定义(DJL Block) |
| SentimentCNN | 情感分析网络(DJL Block) |
| LstmInference | LSTM推理服务 |
| SentimentInference | 情感分析推理服务 |
| LstmTrainer | LSTM训练服务 |
| SentimentTrainer | 情感分析训练服务 |
| DjlModelManager | DJL模型管理器 |

## 技术栈

- DJL 0.28.0
- PyTorch Engine
- ND4J

## 定时任务

| 任务 | Cron | 说明 |
|------|------|------|
| runSentimentAnalysis | 0 0 16 * * 1-5 | 情感分析 |
| runLstmPrediction | 0 30 16 * * 1-5 | LSTM预测 |
| modelEvaluation | 0 30 15 * * 1-5 | 模型评估 |

## 对外API

- `POST /api/ai/sentiment/analyze` - 情感分析
- `POST /api/ai/lstm/predict` - LSTM预测
- `GET /api/ai/model/info` - 模型信息
- `POST /api/ai/train/start` - 启动训练
