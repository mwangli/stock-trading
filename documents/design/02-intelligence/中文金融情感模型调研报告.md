# 中文金融情感分析模型调研报告

## 概述

针对金融领域的情感分析，我们调研了多个中文金融 BERT 预训练模型，这些模型在金融文本上进行了专门的预训练或微调，比通用 BERT 模型更适合金融场景。

## 可选模型清单

### 1. 哈工大中文 BERT（通用，推荐用于金融）⭐

**模型信息**：
- **名称**：`hfl/chinese-bert-wwm-ext`
- **发布机构**：哈工大讯飞联合实验室
- **大小**：400MB
- **特点**：Whole Word Masking，中文 NLP 通用最强
- **适用场景**：通用文本 + 金融文本（配合领域数据微调）
- **HuggingFace**：https://huggingface.co/hfl/chinese-bert-wwm-ext

**优点**：
- 中文理解能力强
- 社区支持好
- 文档齐全
- 可直接使用

**缺点**：
- 非金融专用
- 需要领域数据微调达到最佳效果

---

### 2. 熵简科技 FinBERT（中文金融专用）⭐⭐⭐

**模型信息**：
- **名称**：FinBERT（熵简科技）
- **发布机构**：熵简科技 AI Lab
- **大小**：约 400MB
- **特点**：首个中文金融领域预训练 BERT
- **HuggingFace**：待发布（需从 GitHub 下载）
- **GitHub**：https://github.com/valuesimplex/FinBERT
- **论文**：https://arxiv.org/abs/2006.08097

**技术特点**：
- 基于 BERT 架构
- 在金融语料上使用多任务学习预训练
- 6 种自监督预训练任务
- 在金融下游任务上 F1-score 提升 2~5.7%

**训练语料**：
- 公司报告（2.5 亿词汇）
- 财报电话会议记录（1.3 亿词汇）
- 分析师报告（1.1 亿词汇）
- 总计 4.9 亿词汇金融文本

**下游任务表现**：
- 金融短讯类型分类：F1 提升 5.7%
- 金融行业分类：F1 提升 4.2%
- 金融情绪分类：F1 提升 3.8%
- 金融 NER：F1 提升 3.1%

**下载方式**：
```bash
# 从 GitHub 下载
git clone https://github.com/valuesimplex/FinBERT
# 模型文件在预处理后的目录中
```

---

### 3. 基于 Tushare 的中文金融 BERT ⭐⭐

**模型信息**：
- **名称**：FinBERT-base-on-tushare
- **发布机构**：个人开发者（Ray-gith）
- **大小**：约 400MB
- **特点**：基于 bert-base-chinese 在 Tushare 财经新闻上微调
- **GitHub**：https://github.com/ray-gith/finbert-base-on-tushare

**训练数据**：
- Tushare 财经快讯新闻（2018 年至今）
- 新浪财经频道标签数据
- 100+ 万条财经新闻

**性能**：
- 样本外 AUC ≥ 93%

**下载方式**：
```bash
# 百度网盘下载
链接：https://pan.baidu.com/s/1M9qtgJJqW8eodg7qJrW6yg
提取码：muzr
```

**优点**：
- 专门针对中文财经新闻优化
- 数据源与本项目相同（Tushare）
- 免费下载

**缺点**：
- 个人维护，更新不频繁
- 文档较少

### 4. ProsusAI FinBERT（英文金融）⭐

**模型信息**：
- **名称**：`ProsusAI/finbert`
- **发布机构**：Prosus AI
- **大小**：440MB

- **HuggingFace**：https://huggingface.co/ProsusAI/finbert

**特点**：

**注意**：仅支持英文，不适用于中文场景

## 模型下载与使用

### 哈工大 BERT（推荐）

```bash
cd models/sentiment-analysis
git lfs install
git clone https://huggingface.co/hfl/chinese-bert-wwm-ext
mv chinese-bert-wwm-ext/* .
```

### 熵简 FinBERT

```bash
# 1. 克隆 GitHub 仓库
git clone https://github.com/valuesimplex/FinBERT
cd FinBERT

# 2. 下载预训练模型
# 从论文提供的链接下载 FinBERT 模型文件
# 或使用提供的百度网盘链接（如果有）

# 3. 转换模型格式（如需要）
# 使用 transformers 库加载并保存为 DJL 兼容格式
python convert_model.py --input finbert_model --output ../models/sentiment-analysis
```

### Tushare BERT

```bash
# 1. 从百度网盘下载模型文件
# 链接：https://pan.baidu.com/s/1M9qtgJJqW8eodg7qJrW6yg
# 提取码：muzr

# 2. 解压到模型目录
unzip finbert_tushare.zip -d models/sentiment-analysis/
```

## 配置示例

### application.yml 配置

```yaml
models:
  sentiment:
    # 方案 1：哈工大 BERT（推荐快速启动）
    pretrained-model: hfl/chinese-bert-wwm-ext
    
    # 方案 2：熵简 FinBERT（需要手动下载）
    # pretrained-model: local/finbert-entropy-simplex
    
    # 方案 3：Tushare BERT（需要手动下载）
    # pretrained-model: local/finbert-tushare
    
    model-path: models/sentiment-analysis
    max-sequence-length: 128
    epochs: 5
    batch-size: 16
    learning-rate: 0.00002
    num-labels: 3
    labels:
      - 负面
      - 中性
      - 正面
```
