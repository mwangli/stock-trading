# 交易执行设计

## 1. 模块概述

交易执行模块负责对接券商API，执行买入/卖出订单，并处理订单结果的回调。

## 3. 核心功能

### 3.1 买入委托

```java
public class BuyOrderRequest {
    private String stockCode;       // 股票代码
    private BigDecimal price;        // 委托价格
    private Integer quantity;        // 委托数量 (手)
    private OrderType orderType;     // 订单类型: LIMIT, MARKET
    private BigDecimal targetPrice;  // 目标价
    private BigDecimal stopLossPrice;// 止损价
}

public OrderResult executeBuy(BuyOrderRequest request) {
    // 1. 风控检查
    RiskCheckResult riskCheck = riskService.check(request);
    if (!riskCheck.isPassed()) {
        return OrderResult.fail(riskCheck.getRejectReason());
    }
    
    // 2. 计算买入金额
    BigDecimal amount = request.getPrice().multiply(request.getQuantity());
    
    // 3. 检查资金余额
    BigDecimal available = accountService.getAvailableBalance();
    if (available.compareTo(amount) < 0) {
        return OrderResult.fail("资金不足");
    }
    
    // 4. 发送委托
    String orderId = brokerAPI.sendOrder(request);
    
    // 5. 创建订单记录
    Order order = createOrderRecord(request, orderId);
    
    // 6. 返回结果
    return OrderResult.success(order);
}
```

### 3.2 卖出委托

```java
public class SellOrderRequest {
    private String stockCode;       // 股票代码
    private BigDecimal price;        // 委托价格
    private Integer quantity;        // 委托数量
    private OrderType orderType;     // 订单类型
    private String positionId;      // 持仓ID (关联持仓)
}

public OrderResult executeSell(SellOrderRequest request) {
    // 1. 获取持仓
    Position position = positionService.getById(request.getPositionId());
    
    // 2. T+1 检查
    if (!t1Strategy.canSell(position)) {
        return OrderResult.fail("T+1 限制，次日才能卖出");
    }
    
    // 3. 风控检查
    RiskCheckResult riskCheck = riskService.check(request);
    if (!riskCheck.isPassed()) {
        return OrderResult.fail(riskCheck.getRejectReason());
    }
    
    // 4. 发送委托
    String orderId = brokerAPI.sendOrder(request);
    
    // 5. 创建订单记录
    Order order = createOrderRecord(request, orderId);
    
    // 6. 更新持仓 (冻结部分)
    positionService.freeze(position, request.getQuantity());
    
    return OrderResult.success(order);
}
```

### 3.3 撤单

```java
public CancelResult cancelOrder(String orderId) {
    // 1. 查询订单状态
    Order order = orderRepository.findByOrderId(orderId);
    
    // 2. 检查是否可撤
    if (!order.isCancellable()) {
        return CancelResult.fail("订单不可撤");
    }
    
    // 3. 发送撤单请求
    boolean success = brokerAPI.cancelOrder(orderId);
    
    if (success) {
        // 4. 更新订单状态
        order.setStatus(OrderStatus.CANCELLED);
        orderRepository.save(order);
        
        // 5. 解冻持仓 (如果有)
        if (order.getPositionId() != null) {
            positionService.unfreeze(order.getPositionId(), order.getQuantity());
        }
    }
    
    return CancelResult.success(success);
}
```

| LIMIT | 限价委托 | 指定价格成交 |

## 6. 错误处理

### 6.1 错误码

| 错误码 | 说明 | 处理策略 |
|--------|------|----------|
| E001 | 资金不足 | 降低数量或取消 |
| E002 | 股票停牌 | 取消订单 |
| E003 | 涨跌停 | 调整价格重试 |
| E004 | 数量不足 | 调整数量 |
| E005 | 委托失败 | 重试3次 |
| E006 | 超时 | 查询状态确认 |

### 6.2 重试机制

```java
public String sendOrderWithRetry(OrderRequest request) {
    int maxRetries = 3;
    int retryDelay = 1000; // ms
    
    for (int i = 0; i < maxRetries; i++) {
        try {
            return brokerAPI.sendOrder(request);
        } catch (Exception e) {
            if (i == maxRetries - 1) {
                throw new TradingException("委托失败: " + e.getMessage());
            }
            Thread.sleep(retryDelay * (i + 1));
        }
    }
}
```

### 7.2 回调处理

```java
@PostMapping("/api/broker/callback")
public void handleBrokerCallback(@RequestBody BrokerCallback callback) {
    // 1. 根据订单ID更新订单状态
    Order order = orderRepository.findByOrderId(callback.getOrderId());
    
    // 2. 处理不同状态
    switch (callback.getStatus()) {
        case FILLED:
            handleFilled(order, callback);
            break;
        case PARTIAL_FILLED:
            handlePartialFilled(order, callback);
            break;
        case CANCELLED:
            handleCancelled(order, callback);
            break;
        case REJECTED:
            handleRejected(order, callback);
            break;
    }
}
```
