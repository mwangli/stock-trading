## 2. 概述

### 2.1 模块定位

决策引擎模块是股票交易系统的核心决策组件，负责根据综合选股结果、风控检查结果，生成最终的交易信号（买入/卖出/持有），控制交易流程。

### 2.2 功能摘要

- 根据选股结果生成买入信号
- 根据持仓情况生成卖出信号
- 集成T+1交易策略
- 管理信号有效期

### 2.3 上下游依赖

| 上游模块 | 接口 | 数据 |
|----------|------|------|
| 综合选股 | selectStocks() | 推荐股票列表 |
| 风控管理 | checkBeforeTrade() | 风控检查结果 |

| 下游模块 | 接口 | 数据 |
|----------|------|------|
| 交易执行 | executeTrade() | 交易信号 |

### 3.2 功能详情

#### F-001: 买入决策

**功能描述**: 根据综合选股结果和风控检查，生成买入信号。

**输入**: 推荐股票列表

**处理逻辑**:
1. 遍历推荐股票
2. 检查是否已有持仓
3. 调用风控服务检查
4. 符合条件则生成买入信号

**输出**: 买入信号列表

**业务规则**:
- BUY信号条件：
  - 综合得分 >= 70
  - 预测方向 == "up"
  - 情感 != "negative"
  - 风控检查通过

#### F-002: 卖出决策

**功能描述**: 根据持仓情况和市场条件，生成卖出信号。

**输入**: 当前持仓列表

**处理逻辑**:
1. 遍历持仓股票
2. 检查卖出条件
3. 符合条件则生成卖出信号

**输出**: 卖出信号列表

**业务规则**:
- SELL信号条件（满足任一）：
  - 持仓收益率 <= -3%（止损）
  - 持仓收益率 >= 10%（止盈）
  - 风控熔断触发
  - T+1策略到期

#### F-003: T+1策略

**功能描述**: 实现T+1交易规则。

**业务规则**:
- 当日买入：次日才能卖出
- 当日卖出：当日可以买入

## 6. 详细设计

### 6.1 决策规则

#### 6.1.1 决策规则矩阵

| 选股结果 | 风控检查 | 决策 |
|----------|----------|------|
| 有推荐 | 通过 | BUY |
| 有推荐 | 未通过 | HOLD |
| 无推荐 | 通过 | HOLD |
| 无推荐 | 未通过 | REJECT |

#### 6.1.2 买入条件

```
BUY信号 = 综合得分 >= 70 
        AND 预测方向 == "up" 
        AND 情感 != "negative"
        AND 风控检查通过
```

#### 6.1.3 卖出条件

```
SELL信号 = 持仓收益率 <= -3%
        OR 持仓收益率 >= 10%
        OR 风控熔断触发
        OR T+1 策略到期
```

#### 6.1.4 持有条件

```
HOLD信号 = 不满足BUY条件 
        AND 不满足SELL条件
```

### 6.3 T+1策略实现

```java
public class T1Strategy {
    
    public boolean canSell(String stockCode, LocalDate date) {
        // 查询买入记录
        Optional<Position> position = positionRepository.findByStockCode(stockCode);
        
        if (position.isEmpty()) {
            return false; // 无持仓
        }
        
        // 检查是否T+1
        LocalDate buyDate = position.get().getBuyDate();
        return !buyDate.equals(date); // 不同日期才能卖出
    }
    
    public boolean canBuy(String stockCode, LocalDate date) {
        // 检查今日是否已卖出
        List<Order> todaySellOrders = orderRepository.findByStockCodeAndDate(stockCode, date);
        return todaySellOrders.isEmpty(); // 今日未卖出才能买入
    }
}
```

#### 6.4.3 核心查询SQL示例

**查询推荐股票列表**:
```sql
-- 获取评分>=70且预测上涨的股票
SELECT 
    code, name, score, predict_price, price, increase, permission, selected
FROM stock_info
WHERE score >= 70 
  AND permission = '1'
  AND deleted = '0'
ORDER BY score DESC;
```

**查询当前持仓**:
```sql
-- 获取持仓中的股票(用于卖出决策)
SELECT 
    p.stock_code, p.stock_name, p.quantity, p.avg_cost, 
    p.current_price, p.unrealized_pnl, p.unrealized_pnl_ratio,
    p.open_date, s.price as current_market_price
FROM position p
LEFT JOIN stock_info s ON p.stock_code = s.code
WHERE p.status = 'HOLDING';
```

**查询当日交易次数**:
```sql
-- 统计当日已完成的交易(用于T+1判断)
SELECT 
    code, COUNT(*) as trade_count,
    SUM(CASE WHEN type = '买入' THEN number ELSE 0 END) as buy_count,
    SUM(CASE WHEN type = '卖出' THEN number ELSE 0 END) as sell_count
FROM order_info
WHERE date = CURDATE()
  AND status = '已成交'
GROUP BY code;
```

**查询模型评分**:
```sql
-- 获取股票对应的LSTM模型评分
SELECT code, name, test_deviation, score, status
FROM model_info
WHERE code IN ('600519', '000001', ...)
  AND status = 'ACTIVE';
```
