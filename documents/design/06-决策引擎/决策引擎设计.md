# 决策引擎设计

## 1. 模块概述

决策引擎负责根据综合选股结果、风控检查结果，生成最终的交易信号（买入/卖出/持有）。

## 2. 决策逻辑

### 2.1 决策规则矩阵

| 选股结果 | 风控检查 | 决策 |
|----------|----------|------|
| 有推荐 | 通过 | 买入 |
| 有推荐 | 未通过 | 持有 |
| 无推荐 | 通过 | 持有 |
| 无推荐 | 未通过 | 卖出 |

### 2.2 买入条件

```
买入信号 = 综合得分 >= 70 
        AND 预测方向 == "up" 
        AND 情感 != "negative"
        AND 风控检查通过
```

### 2.3 卖出条件

```
卖出信号 = 持仓收益率 <= -3%
        OR 持仓收益率 >= 10%
        OR 风控熔断触发
        OR T+1 策略到期
```

### 2.4 持有条件

```
持有信号 = 不满足买入条件 
        AND 不满足卖出条件
```

## 4. T+1 策略

### 4.1 策略规则

- **当日买入**: 次日才能卖出
- **当日卖出**: 当日可以买入

### 4.2 实现

```java
public class T1Strategy {
    
    public boolean canSell(String stockCode, LocalDate date) {
        // 查询买入记录
        Optional<Position> position = positionRepository.findByStockCode(stockCode);
        
        if (position.isEmpty()) {
            return false; // 无持仓
        }
        
        // 检查是否T+1
        LocalDate buyDate = position.get().getBuyDate();
        return !buyDate.equals(date); // 不同日期才能卖出
    }
    
    public boolean canBuy(String stockCode, LocalDate date) {
        // 检查今日是否已卖出
        List<Order> todaySellOrders = orderRepository.findByStockCodeAndDate(stockCode, date);
        return todaySellOrders.isEmpty(); // 今日未卖出才能买入
    }
}
```

## 5. 数据结构

### 5.1 决策请求

```java
public class DecisionRequest {
    private LocalDate tradingDate;       // 交易日期
    private List<String> recommendations; // 推荐的股票列表
    private List<Position> currentPositions; // 当前持仓
}
```

### 5.2 交易信号

```java
public class TradingSignal {
    private String signal;                // buy/sell/hold
    private String stockCode;            // 股票代码
    private BigDecimal price;             // 信号价格
    private BigDecimal quantity;         // 数量
    private BigDecimal targetPrice;       // 目标价
    private BigDecimal stopLossPrice;    // 止损价
    private String reason;               // 生成原因
    private LocalDateTime generatedAt;   // 生成时间
    private LocalDateTime expiresAt;     // 失效时间
}
```

### 5.3 决策响应

```java
public class DecisionResponse {
    private List<TradingSignal> signals; // 交易信号列表
    private RiskCheckResult riskCheck;   // 风控检查结果
    private Map<String, Object> metadata; // 附加信息
}
```

## 6. 定时任务

### 6.1 任务配置

| 任务名称 | 执行时间 | 频率 | 说明 |
|----------|----------|------|------|
| 开盘前决策 | 09:15 | 每日 | 生成当日买入信号 |
| 盘中决策 | 10:00, 11:00, 14:00 | 盘中每小时 | 动态调整 |
| 收盘决策 | 14:30 | 每日 | 生成卖出信号 |

### 6.2 Cron 表达式

```java
@Scheduled(cron = "0 15 9 * * *")      // 每日 09:15
@Scheduled(cron = "0 0 10,11,14 * * *") // 10:00, 11:00, 14:00
@Scheduled(cron = "0 30 14 * * *")    // 每日 14:30
```

## 7. 与其他模块集成

### 7.1 依赖模块

| 模块 | 接口 | 用途 |
|------|------|------|
| 综合选股 | StockSelectionService | 获取推荐股票 |
| 风控管理 | RiskManagementService | 风控检查 |
| 交易执行 | TradingExecutionService | 执行交易 |

### 7.2 服务编排

```java
@Service
public class DecisionEngineService {
    
    public DecisionResponse makeDecision(DecisionRequest request) {
        // 1. 获取推荐股票
        List<String> recommendations = getRecommendations();
        
        // 2. 获取当前持仓
        List<Position> positions = getCurrentPositions();
        
        // 3. 遍历持仓，检查卖出
        List<TradingSignal> sellSignals = checkSell(positions);
        
        // 4. 如果无持仓且有推荐，检查买入
        List<TradingSignal> buySignals = Collections.emptyList();
        if (positions.isEmpty() && !recommendations.isEmpty()) {
            buySignals = checkBuy(recommendations);
        }
        
        // 5. 合并信号
        List<TradingSignal> allSignals = new ArrayList<>();
        allSignals.addAll(sellSignals);
        allSignals.addAll(buySignals);
        
        return DecisionResponse.builder()
            .signals(allSignals)
            .build();
    }
}
```
