# LSTM 预测服务设计

## 1. 模块概述

LSTM 预测模块负责使用长短期记忆网络（LSTM）对股票价格进行预测，为选股系统提供技术因子。

## 2. 技术架构

### 2.1 技术栈

| 组件 | 技术选型 | 版本 |

| 深度学习框架 | TensorFlow / Keras | 2.15.x |
| 数值计算 | NumPy | 1.x |
| 数据处理 | Pandas | 2.x |
| 数据预处理 | Scikit-learn | 1.x |
| Web框架 | FastAPI | 0.109.x |

## 3. 服务设计

### 3.1 LSTMService 类

```python
class LSTMService:
    """LSTM-based stock prediction service"""

    def __init__(self):

        self._model = None

        """预测次日价格"""

    def predict_batch(self, stock_code: str, data: List[Dict], days: int = 5) -> Dict:
        """批量预测多日"""
        pass

    def get_model_info(self) -> Dict:

```

### 3.2 核心方法

#### predict(stock_code, data)

单股票价格预测。

| 参数 | 类型 | 说明 |
|------|------|------|
| stock_code | str | 股票代码 |
| data | List[Dict] | 历史K线数据，需至少60条 |

**data 格式**:
```json
[
    {"date": "2024-01-01", "open": 10.5, "high": 10.8, "low": 10.4, "close": 10.6, "volume": 1000000},
    ...
]
```

| 返回 | 类型 | 说明 |
|------|------|------|
| stock_code | str | 股票代码 |
| predicted_price | float | 预测价格 |
| current_price | float | 当前价格 |
| change | float | 预测涨跌额 |
| change_percent | float | 预测涨跌幅 |
| direction | str | up/down/neutral |
| confidence | float | 置信度 (0-1) |
| is_trained | bool | 模型是否训练过 |

#### predict_batch(stock_code, data, days)

多日价格预测。

| 参数 | 类型 | 说明 |
|------|------|------|
| stock_code | str | 股票代码 |
| data | List[Dict] | 历史K线数据 |
| days | int | 预测天数 (默认5) |

| 返回 | 类型 | 说明 |
|------|------|------|
| stock_code | str | 股票代码 |
| predictions | List[Dict] | 预测结果列表 |
| is_trained | bool | 模型是否训练过 |

## 4. 模型设计

### 4.1 网络结构

```
┌─────────────────────────────────────────────────────────────┐
│                    LSTM Model Architecture                   │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  Input: (batch_size, 60, 5)    # 60天历史, 5个特征        │
│         │                                                │
│         ▼                                                │
│  ┌─────────────────────────────────────────────────┐      │
│  │  LSTM(50, return_sequences=True)              │      │
│  │  Dropout(0.2)                                  │      │
│  └─────────────────────────────────────────────────┘      │
│         │                                                │
│         ▼                                                │
│  ┌─────────────────────────────────────────────────┐      │
│  │  LSTM(50, return_sequences=False)             │      │
│  │  Dropout(0.2)                                  │      │
│  └─────────────────────────────────────────────────┘      │
│         │                                                │
│         ▼                                                │
│  ┌─────────────────────────────────────────────────┐      │
│  │  Dense(25, activation='relu')                  │      │
│  └─────────────────────────────────────────────────┘      │
│         │                                                │
│         ▼                                                │
│  ┌─────────────────────────────────────────────────┐      │
│  │  Dense(1)                                       │      │
│  └─────────────────────────────────────────────────┘      │
│         │                                                │
│         ▼                                                │
│  Output: (batch_size, 1)       # 次日收盘价预测          │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 4.2 输入特征

| 序号 | 特征 | 说明 |
|------|------|------|
| 1 | Open | 开盘价 |
| 2 | High | 最高价 |
| 3 | Low | 最低价 |
| 4 | Close | 收盘价 |
| 5 | Volume | 成交量 |

### 4.3 数据预处理

1. **归一化**: 使用 MinMaxScaler 将数据缩放到 (0, 1) 范围
2. **序列生成**: 将60天数据生成一个输入序列
3. **反归一化**: 预测结果逆变换为实际价格

## 5. 预测流程

```
输入: 股票代码 + 60天OHLCV数据

┌─────────────────────────────────────────────────────────────┐
```

## 7. 错误处理

### 7.1 异常类型

| 异常 | 说明 | 处理 |
|------|------|------|
| InsufficientDataError | 数据不足 | 返回错误信息，要求至少60条 |
| ModelLoadError | 模型加载失败 | 使用 fallback 模型 |
| PredictionError | 预测失败 | 返回错误信息 |

### 7.2 Fallback 模式

当训练模型不存在时，使用随机初始化的模型，置信度降低到 0.5。

```json
{
    "confidence": 0.5,
    "is_trained": false,
    "model_status": "fallback"
}
```
