# databus 模块 - 数据总线

## 模块说明

databus 模块负责数据采集、存储、缓存和分发，是整个系统的基础数据层。

## 包含功能

| 序号 | 功能 | 说明 |
|------|------|------|
| 1 | 股票列表同步 | 东方财富API获取A股股票列表 |
| 2 | 历史K线获取 | Baostock/东方财富获取历史数据 |
| 3 | 实时行情 | 实时涨跌幅、成交量 |
| 4 | 新闻采集 | 财经新闻获取 |
| 5 | 数据存储 | MySQL + MongoDB |
| 6 | 缓存管理 | Redis热点数据 |

## 核心类

| 类名 | 说明 |
|------|------|
| EastMoneyClient | 东方财富HTTP客户端 |
| BaoStockClient | Baostock HTTP客户端 |
| StockCollector | 股票数据采集器 |
| NewsCollector | 新闻采集器 |
| QuoteCollector | 实时行情采集器 |
| StockRepository | MyBatis数据仓储 |
| PriceRepository | MongoDB价格仓储 |
| CacheManager | Redis缓存管理 |

## 技术栈

- OkHttp (HTTP客户端)
- MyBatis-Plus (MySQL)
- Spring Data MongoDB
- Spring Data Redis

## 定时任务

| 任务 | Cron | 说明 |
|------|------|------|
| syncStockList | 0 0 9 * * 1-5 | 同步股票列表 |
| syncHistoricalData | 0 0 9 * * 1-5 | 同步历史数据 |
| syncRealtimeQuotes | 0 */5 * * * * | 实时行情(每5分钟) |
| collectNews | 0 30 15 * * 1-5 | 采集新闻 |

## 对外API

- `GET /api/data/stock/list` - 获取股票列表
- `GET /api/data/stock/prices` - 获取历史K线
- `GET /api/data/quote/realtime` - 获取实时行情
- `GET /api/data/news` - 获取财经新闻
