## 1. 模块概述

数据采集模块是整个系统的基础数据层，负责从**证券平台**获取股票数据，并存储到 MongoDB 数据库中。

## 3. 数据存储需求

### 3.1 MongoDB 存储

| 集合名 | 存储内容 | 主要字段 |
|--------|----------|----------|
| stock_prices | K 线价格数据 | code, name, date, openPrice, highPrice, lowPrice, closePrice, volume, amount |
| stock_info | 股票基本信息 | code, name, listDate, totalShares, circulatingShares |

## 4. 接口需求

### 4.1 对外提供接口

```java
// 股票数据服务
StockDataService
  - getHistoryPrices(String code): List<StockPrice>
  - saveStockPrices(List<StockPrice> prices): void
  - syncHistoricalData(String code, startDate, endDate): int
  - getAllStocks(): List<StockInfo>
  - getLatestPrice(String code): StockPrice

// Repository 层
PriceRepository
  - findByCodeOrderByDateAsc(String code): List<StockPrice>
  - findByCodeAndDateBetween(code, start, end): List<StockPrice>
  - existsByCodeAndDate(code, date): boolean
  
StockRepository
  - findByCode(String code): Optional<StockInfo>
  - findAllByOrderByCodeAsc(): List<StockInfo>
```

### 4.2 服务消费者

- **AI 模型模块**: 获取 K 线数据用于 LSTM 训练和预测
- **策略模块**: 获取股票列表和价格数据用于选股

## 5. 性能需求

| 指标 | 要求 |
|------|------|
| 数据查询响应 | 95% 查询 < 200ms |
| 数据存储延迟 | 采集后 5 分钟内完成存储 |
| 定时任务执行 | 成功率 ≥ 99% |
| 数据完整性 | 100% |

## 6. 容错需求

- 网络失败自动重试 3 次
- 单支股票采集失败不影响其他股票
- 数据采集失败要有详细日志记录
- MongoDB 连接失败自动重连

## 9. 数据字典

### StockPrice 实体

| 字段 | 类型 | 说明 |
|------|------|------|
| id | String | MongoDB 主键 |
| code | String | 股票代码 |
| date | LocalDate | 交易日期 |
| openPrice | BigDecimal | 开盘价 |
| highPrice | BigDecimal | 最高价 |
| lowPrice | BigDecimal | 最低价 |
| closePrice | BigDecimal | 收盘价 |
| volume | BigDecimal | 成交量（手） |
| amount | BigDecimal | 成交额（元） |

### StockInfo 实体

| 字段 | 类型 | 说明 |
|------|------|------|
| id | Long | MySQL 主键 |
| code | String | 股票代码 |
| name | String | 股票名称 |
| market | String | 市场代码 |
| price | BigDecimal | 当前价格 |
| changeAmount | BigDecimal | 涨跌额 |
| changePercent | BigDecimal | 涨跌幅 |
| totalMarketValue | BigDecimal | 总市值 |
| turnoverRate | BigDecimal | 换手率 |
| volumeRatio | BigDecimal | 量比 |
| industryCode | Integer | 行业代码 |
| createTime | LocalDateTime | 创建时间 |
| updateTime | LocalDateTime | 更新时间 |
