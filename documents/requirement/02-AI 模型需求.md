# 02-AI 模型需求文档

## 文档信息

| 项目 | 内容 |
|------|------|
| 模块 | AI 模型模块 (Models) |
| 包路径 | com.stock.modelService |
| 版本 | 1.0.0 |

## 1. 模块概述

AI 模型模块是系统的智能核心，负责股票价格预测和财经新闻情感分析。

## 2. 功能需求

### 2.1 LSTM 价格预测

| 需求 ID | 需求描述 | 优先级 | 验收标准 |
|--------|----------|--------|----------|
| MODELS-REQ-001 | 基于历史 K 线预测次日涨跌概率 | P0 | 预测准确率 > 55% |
| MODELS-REQ-002 | 支持批量预测所有股票 | P0 | 能处理全市场股票 |
| MODELS-REQ-003 | 模型加载和初始化 | P0 | 启动时自动加载模型 |
| MODELS-REQ-004 | 预测结果缓存 | P1 | 相同输入返回缓存结果 |
| MODELS-REQ-005 | 模型版本管理 | P1 | 支持模型热更新 |

### 2.2 情感分析

| 需求 ID | 需求描述 | 优先级 | 验收标准 |
|--------|----------|--------|----------|
| MODELS-REQ-006 | 对财经新闻进行情感分析 | P0 | 输出利好/利空/中性 |
| MODELS-REQ-007 | 识别新闻中的股票代码 | P0 | 准确率 > 85% |
| MODELS-REQ-008 | 情感得分量化 (-1 到 1) | P0 | 得分连续可比较 |
| MODELS-REQ-009 | 批量处理新闻数据 | P0 | 支持批量分析 |

### 2.3 模型训练

| 需求 ID | 需求描述 | 优先级 | 验收标准 |
|--------|----------|--------|----------|
| MODELS-REQ-010 | 支持 LSTM 模型训练 | P1 | 能训练新模型 |
| MODELS-REQ-011 | 训练数据预处理 | P1 | 数据标准化、归一化 |
| MODELS-REQ-012 | 训练过程监控 | P1 | 记录损失和准确率 |

## 3. 性能需求

| 指标 | 要求 |
|------|------|
| 单次预测响应时间 | < 100ms |
| 批量预测 (全市场) | < 30 分钟 |
| 情感分析速度 | > 100 条/分钟 |
| 模型加载时间 | < 10 秒 |

## 4. 数据需求

### 4.1 输入数据

- LSTM: 60 日 K 线数据 (开高低收成交量)
- 情感分析：新闻标题 + 内容

### 4.2 输出数据

- LSTM: 涨的概率 (0-1), 预测价格变化率
- 情感分析：情感得分 (-1 到 1), 关联股票代码

## 5. 接口需求

```java
// LSTM 预测接口
LstmPrediction predict(String stockCode, List<KlineData> history);
Map<String, LstmPrediction> batchPredict(List<String> stockCodes);

// 情感分析接口
SentimentResult analyzeSentiment(String newsContent);
List<SentimentResult> batchAnalyze(List<String> newsContents);

// 模型管理接口
void reloadModel();
ModelInfo getModelInfo();
```

## 6. 依赖关系

### 6.1 外部依赖

- Deep Java Library (DJL)
- PyTorch 引擎
- HuggingFace Tokenizers (BERT)

### 6.2 内部依赖

- 数据采集模块：提供 K 线数据和新闻数据
- 策略模块：消费预测和情感分析结果
