## 1. 模块概述

策略模块是系统的决策核心，包含两类策略：
1. **选股策略**：基于 AI 模型预测结果生成股票评分和买入信号 (天级)
2. **T+1 卖出策略**：基于分钟级价格走势的实时卖出决策 (分钟级)

## 3. 算法需求

### 3.1 综合得分计算 (选股策略)

```
综合得分 = LSTM 因子得分 × 0.6 + 情感因子得分 × 0.4

LSTM 因子得分 = (涨的概率 - 0.5) × 2  [归一化到 0-1]
情感因子得分 = (情感得分 + 1) / 2  [从 -1~1 映射到 0-1]
```

### 3.2 股票排名

- 按综合得分降序排序
- 输出所有股票的排名和得分

### 3.3 信号生成规则

```
买入信号:
- 综合排名前 3 且无持仓 → 买入信号
- 信号强度：第 1 名=100, 第 2 名=80, 第 3 名=60

卖出信号 (持仓股排名):
- 持仓股跌出前 10 → 卖出信号
- 信号强度 = 50 + (10 - 排名) × 5

T+1 卖出信号 (分钟级):
- 移动止损触发 → 立即卖出
- RSI 超买 + 量价背离 → 卖出 (得分≥60)
- 布林带连续突破 → 卖出 (得分≥80)
- 14:57 强制截止 → 市价卖出
```

### 3.4 T+1 卖出策略算法

**移动止损计算:**
```
if 当前价 > 最高价:
    最高价 = 当前价
    止损线 = 最高价 × (1 - 回撤容忍度)

回撤容忍度:
- 基础值：2%
- 高波动股票 (日振幅>5%)：3%
- 低波动股票 (日振幅<3%)：1%

触发条件：当前价 <= 止损线
```

**RSI 超买计算:**
```
RS = 平均上涨幅度 (14) / 平均下跌幅度 (14)
RSI = 100 - [100 / (1 + RS)]

触发条件：RSI(14) > 75 AND 当前价 > 开盘价 × 1.02
```

**成交量背离计算:**
```
当前价是最近 5 分钟最高 AND
最近 1 分钟成交量 < 前 5 分钟均量 × 0.8

触发条件：量价背离确认
```

**布林带突破计算:**
```
中轨 = MA(close, 20)
上轨 = 中轨 + 2 × STD(close, 20)

触发条件：
当前价 > 上轨 × 1.005 AND 连续 3 分钟在上轨之上
```

**综合得分计算 (T+1 策略):**
```
指标           权重    触发时得分
移动止损       40%     40
RSI 超买        20%     20
成交量背离     20%     20
布林带上轨     20%     20
总分           100%    100

总分 = Σ(指标触发状态 × 指标权重)
```

**动态阈值设计:**
```
当日涨幅          卖出阈值    逻辑
>3%              50%        已有不错收益，可以更积极
1% - 3%          60%        正常情况
<1%              80%        收益有限，需更强信号
```
