## 1. 系统初始化

### 1.1 更新系统

```bash
# 更新软件包
sudo apt update && sudo apt upgrade -y

# 安装基础工具
sudo apt install -y curl wget git vim htop net-tools
```

### 1.2 创建工作目录

```bash
# 创建项目目录
sudo mkdir -p /opt/stock-trading
sudo chown -R ubuntu:ubuntu /opt/stock-trading
```

---

## 2. 网络配置

### 2.1 开放端口

```bash
# 开放必要端口
sudo ufw allow 22/tcp    # SSH
sudo ufw allow 8000/tcp  # 前端
sudo ufw allow 8080/tcp  # 后端
sudo ufw allow 6033/tcp  # MySQL
sudo ufw allow 6379/tcp  # Redis
sudo ufw allow 27017/tcp # MongoDB

# 启用防火墙
sudo ufw enable

# 查看状态
sudo ufw status
```

### 2.2 端口说明

| 端口 | 服务 | 说明 |
|------|------|------|
| 22 | SSH | 远程管理 |
| 8000 | Frontend | Web界面 |
| 8080 | Backend | API服务 |
| 6033 | MySQL | 数据库 |
| 6379 | Redis | 缓存 |
| 27017 | MongoDB | 文档数据库 |

## 3. 时区配置

```bash
# 设置时区
sudo timedatectl set-timezone Asia/Shanghai

# 验证
timedatectl
```

## 4. 防火墙配置（可选）

如果使用腾讯云安全组，在控制台开放以下端口：

- 8000, 8080, 6033, 6379, 27017

---

## 5. SSH配置（可选）

```bash
# 生成SSH密钥
ssh-keygen -t ed25519 -C "your_email@example.com"

# 复制公钥到服务器
ssh-copy-id ubuntu@124.220.36.95

# 配置SSH无密码登录
# 编辑 /etc/ssh/sshd_config
PasswordAuthentication no
PubkeyAuthentication yes

sudo systemctl restart sshd
```
