## 1. 测试范围

### 1.1 测试目标
验证LSTM预测模块的所有功能是否按需求正常工作，包括：
- LSTM模型训练
- 数据预处理和归一化
- 单股票价格预测
- 批量价格预测
- **预测结果真实写入 Redis 缓存**
- **模型信息真实写入 MySQL**

### 1.2 测试环境
- **JDK**: 17
- **框架**: Deeplearning4J
- **测试框架**: Spring Boot Test + JUnit 5
- **数据库**: MySQL 8.0 (直连远程Docker实例) + MongoDB 6.0 (直连远程Docker实例)
- **缓存**: Redis (Embedded)
- **数据管理**: Database Rider

### 1.3 测试原则
| 原则 | 说明 |
|------|------|
| 真实性 | 使用真实数据库，验证数据真实写入 |
| 隔离性 | @Transactional 自动回滚，不污染环境 |
| 可追溯 | 每个步骤都可验证数据状态 |
| 全链路 | 从数据读取到预测结果完整穿透 |
