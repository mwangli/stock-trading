## 4. 数据库验证方法

### 4.1 标准验证模式

```java
// 1. 验证数据存在
assertDataExists("stock_info", "code = '000001'");

// 2. 验证数据不存在
assertDataNotExists("stock_info", "code = '999999'");

// 3. 验证记录数
assertRowCount("stock_info", 10);

// 4. 验证字段值
String name = queryValue("SELECT name FROM stock_info WHERE code = ?", String.class, "000001");
assertThat(name).isEqualTo("平安银行");

// 5. MongoDB 验证
assertMongoDataExists("stock_prices", "code", "000001");
```

### 4.2 验证点清单

每个测试用例必须包含以下验证点：

- [ ] **V1 数据源验证**: 输入数据/外部API响应正确

- [ ] **V3 转换验证**: 数据转换/映射正确
