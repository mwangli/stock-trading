## 3. 系统测试

### 3.1 端到端测试

**测试场景：LSTM 预测流程**
```
1. 从 MongoDB 读取 K 线数据
2. 数据预处理
3. LSTM 模型推理
4. 结果缓存到 Redis
5. 输出预测结果
```

**测试场景：情感分析流程**
```
1. 从 MySQL 读取新闻
2. 文本清洗
3. BERT 推理
4. 情感得分计算
5. 股票代码识别
6. 输出分析结果
```

## 6. 模型准确率测试

### 6.1 LSTM 预测准确率

**测试方法:**
1. 使用历史数据回测
2. 比较预测和实际涨跌
3. 计算准确率

**验收标准:**
- 预测准确率 > 55%

### 6.2 情感分析准确率

**测试方法:**
1. 人工标注测试集
2. 比较模型预测和人工标注
3. 计算准确率

**验收标准:**
- 情感分析准确率 > 80%
- 股票代码识别准确率 > 85%
