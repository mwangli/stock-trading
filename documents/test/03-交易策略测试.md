## 3. 系统测试

### 3.1 选股策略系统测试

**测试场景：综合选股流程**
```
1. 从 AI 模块获取预测结果
2. 从数据模块获取股票列表
3. 因子计算和标准化
4. 综合得分计算
5. 股票排名排序
6. Top N 筛选
7. 保存排名结果
```

### 3.2 T+1 卖出策略系统测试

**测试场景：完整卖出决策流程**
```
1. 接收分钟级行情数据
2. 更新当日最高价和止损线
3. 计算 RSI、成交量、布林带指标
4. 检查各策略触发条件
5. 计算综合得分
6. 对比动态阈值
7. 生成卖出决策
8. 发送到执行模块
```

**测试场景：移动止损触发**
```
1. 买入股票 (成本价 10.00)
2. 价格上涨到 10.50 (止损线更新为 10.29)
3. 价格回落到 10.29
4. 验证触发移动止损卖出信号
```

**测试场景：RSI 超买触发**
```
1. 模拟 RSI 持续上升到 76
2. 价格上涨超过开盘价 2%
3. 验证触发 RSI 超买卖出信号
```

**测试场景：量价背离触发**
```
1. 价格创 5 分钟新高
2. 成交量萎缩到前均量的 75%
3. 验证触发量价背离卖出信号
```

**测试场景：尾盘强制卖出**
```
1. 时间到达 14:57
2. 验证无论指标状态如何，生成强制卖出信号
```

**测试场景：策略熔断**
```
1. 模拟连续 3 次卖出后价格继续上涨>2%
2. 验证策略暂停 1 小时
3. 验证 1 小时后策略恢复
```

## 6. 策略验证测试

### 6.1 选股策略回测

**测试方法:**
1. 使用历史数据回测
2. 模拟选股和交易
3. 计算收益率和胜率

**验收标准:**
- 回测收益率 > 基准指数
- 胜率 > 50%

### 6.2 T+1 卖出策略回测

**测试方法:**
1. 使用历史分钟级数据回测
2. 模拟完整 T+1 交易流程
3. 对比策略卖出价和当日最高价

**核心指标:**
- **高点捕获率**: 卖出价 / 当日最高价 > 75%
- **胜率**: 卖出价 > 次日开盘价 > 60%
- **平均优化收益**: 相比随机卖出的超额收益 > 0.5%
- **最大连续失败**: < 5 次

### 6.3 A/B 测试方案

**对照组:** 尾盘随机时间卖出 (14:30-14:57 随机)
**实验组:** 策略触发卖出

**测试周期:** 至少 20 个交易日
**样本量:** 每只股票至少 30 次交易

### 6.4 权重敏感性测试

**测试方法:**
1. 调整各指标权重
2. 比较卖出决策差异
3. 验证权重影响

**测试场景:**
- 移动止损优先：50%/20%/20%/10%
- 均衡配置：40%/20%/20%/20%
- 技术指标优先：30%/30%/20%/20%
