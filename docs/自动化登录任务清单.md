# 自动化登录任务清单

> 目标：从零到完整实现「手机验证页点击获取验证码 → 滑块弹出 → 滑块验证通过 → 短信验证码 → 跳转登录页 → 登录 → Token 获取」全链路。
>
> 原则：每个步骤都是可执行、可验证的，必须验收通过才进入下一步。
