## 模块对应关系

| 模块 | 包路径 | 职责概要 |
|------|--------|----------|
| 数据采集 | com.stock.dataCollector | 股票列表、K 线、新闻采集与存储 |
| 模型服务 | com.stock.modelService | LSTM 预测、情感分析、模型训练 |
| 策略分析 | com.stock.strategyAnalysis | 双因子选股、T+1 卖出策略、信号生成 |

## 说明

- 每个模块目录下的**任务.md** 记录所有已完成和待完成的任务事项。
- 历史实施计划（如动态任务调度、WebSocket、订单通知等）已移除，仅保留长期有效的需求与设计。
- 若需查阅旧版文档或实施记录，可查看 Git 历史。
