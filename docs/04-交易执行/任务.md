# 04-交易执行任务清单

> 模块：交易执行模块 (Trading Executor)
> 包路径：`com.stock.tradingExecutor`
> 版本：4.0.0

---

## 已完成任务

| # | 任务 | 说明 |
|---|------|------|
| 1 | 动态任务调度 | `JobConfig` 表 + `JobSchedulerService`，支持 Cron 可变、启停、手动触发、状态回写 |
| 2 | WebSocket 实时推送 | `/ws/notifications` 和 `/ws/logs` 双通道，广播日志与通知 |
| 3 | 订单通知 | `OrderNotificationEvent` + `@Async NotificationListener`，订单状态变更推送 |
| 4 | 价格监控器 | `PriceMonitor` 组件，监控价格决定订单执行时机 |
| 5 | 订单轮询器 | `OrderPoller` 组件，跟踪订单执行状态 |
| 6 | 券商适配器 | `BrokerAdapter` 接口，抽象券商 API 差异（当前支持中信证券） |
| 7 | 交易时间检查器 | `TradingTimeChecker` 组件，避免非交易时段下单 |
| 8 | 手续费计算器 | `FeeCalculator` 组件，含佣金、印花税、过户费计算 |
| 9 | 风控系统 | 仓位上限、单次买入限额、止损线检查 |
| 10 | 模拟订单执行 | 完整买入/卖出模拟，含成本记录和持仓更新 |
| 11 | 滑块验证码破解 | `CaptchaService`，支持中信证券登录滑块自动识别（已迁移至 05-自动登录模块） |
