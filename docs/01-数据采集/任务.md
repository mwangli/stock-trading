# 01-数据采集任务清单

> 模块：数据采集模块 (Data Collector)
> 包路径：`com.stock.dataCollector`
> 版本：3.2.0

---

## 已完成任务

| # | 任务 | 说明 |
|---|------|------|
| 1 | 多模块架构搭建 | 作为独立 Maven 子模块，标准分层（client/service/repository/entity/scheduled） |
| 2 | 股票列表同步 | `StockListSyncService` 从证券平台同步全量股票列表 |
| 3 | 智能增量同步 | 基于 `lastSyncDate` 自动计算缺失日期范围，支持断点续传和空洞填补 |
| 4 | K 线数据采集 | 每日增量 8 线程并行采集，冷启动全量 8 线程线程池 |
| 5 | 新闻与公告采集 | `StockNewsScheduler`，支持全量/增量、多线程、批量写入 |
| 6 | 证券平台 403 处理 | 支持 HTTP/SOCKS5 代理配置，解决云服务器 IP 被拦截问题 |
| 7 | BulkOperations upsert 写入优化 | 批量写入 StockPrice 已最优 |
