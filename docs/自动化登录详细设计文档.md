## 1. 系统概述

### 1.1 目标

实现中信证券交易系统的自动化登录流程，包括：
- 数学运算验证码自动计算 / 图片验证码人工介入
- **网易云盾（NECaptcha）滑块验证码**自动处理
- 手机短信验证码处理（新设备/新环境）
- Token 获取与自动保活

### 1.2 技术方案

使用 Selenium WebDriver 驱动真实 Chrome 浏览器，让网易云盾的 JS 在真实环境中自动执行（生成 `fp`/`cb` 等指纹参数），通过图像识别计算滑块距离，模拟人类拖动完成验证。

### 1.3 登录 URL

```
https://weixin.citicsinfo.com/tztweb/deal/index.html#!/account/login.html
```

### 1.4 页面类型

| 页面 | URL 特征 | 说明 |
|------|---------|------|
| `login.html` | `#!/account/login.html` | 标准登录（账号+密码+验证码） |
| `activePhone.html` | `#!/account/activePhone.html` | 新设备手机验证（SMS + 可能的滑块） |

## 2. 运行模式

### 2.1 常驻浏览器模式（推荐，生产环境）

浏览器启动后不关闭，持续运行。通过 API 接口触发登录，`AutoLoginMaintenanceJob` 每 30 分钟自动检查登录状态并重登。

```bash
# 启动后端，浏览器自动常驻
cd backend && mvn spring-boot:run

# API 操作
POST /api/auto-login/login?username=XXX&password=XXX
GET  /api/auto-login/status
POST /api/auto-login/quit
```

### 2.2 分步调试模式（开发环境）

通过 `BrowserApiController` 的细粒度 API 逐步执行和验证每个环节。

### 2.3 JUnit 测试模式

通过测试类逐步骤验证，浏览器在测试结束后关闭。

## 4. 详细设计

### 阶段一：浏览器环境初始化

#### 4.1 启动 Chrome 浏览器

**负责组件**：`BrowserSessionManager.startBrowser()`

**流程**：

```
1. 配置 ChromeOptions
   ├── --no-sandbox
   ├── --disable-dev-shm-usage
   ├── --disable-blink-features=AutomationControlled
   ├── --window-size=1920,1080
   └── --headless（按配置决定）

2. 创建 WebDriver 实例
   ├── chrome.remote.url 有值 → new RemoteWebDriver(remoteUrl, options)
   └── chrome.remote.url 为空 → WebDriverManager 自动下载 + new ChromeDriver(options)

3. 注入浏览器指纹（injectFingerprint）
   ├── navigator.webdriver = undefined
   ├── languages = ["zh-CN", "zh", "en"]
   ├── platform = "Win32"
   ├── hardwareConcurrency = 8
   └── deviceMemory = 8

4. 访问登录页面
```

**验证**：`driver != null && currentUrl 包含 "login.html" 或 "activePhone.html"`

### 阶段二：页面类型检测与 iframe 切换

#### 4.2 检测页面类型

**负责组件**：`AutoLoginService.performLogin()`

```
String url = driver.getCurrentUrl();
if (url.contains("activePhone.html")) {
    → handlePhoneVerification()  // 手机验证流程

    → 标准登录流程
```

#### 4.3 切换到登录表单 iframe

**负责组件**：`BrowserSessionManager.ensureLoginFrame()`

登录表单嵌套在 iframe 中，需递归遍历所有 frame 找到包含密码输入框的 frame：

```
1. 在当前 frame 查找 input[type="password"]
2. 如果找到 → 返回（已在正确的 frame 中）
3. 如果未找到 → 遍历所有子 iframe：
   a. switchTo().frame(iframe)
   b. 递归查找
   c. 未找到则 switchTo().parentFrame() 回退
```

### 阶段三：表单填写

#### 4.4 输入账号

**负责组件**：`LoginPageHandler.inputAccount()` → `BrowserSessionManager.inputAccount()`

**定位策略**（按优先级）：
1. `input[id="account"]` 或 `input[name="account"]`
2. `input[placeholder*="账号"]` 或 `input[placeholder*="资金"]`

**输入方式**：逐字符输入，每字符间隔 80-140ms 随机延迟，输入后触发 `input` 事件。

#### 4.5 输入密码

**负责组件**：`LoginPageHandler.inputPassword()`

**定位策略**：`input[type="password"]` 或 `input[id="password"]`

**输入方式**：同账号输入，逐字符 + 随机延迟。

#### 4.6 处理验证码

**负责组件**：`AutoLoginService.handleCaptchaStep()`

```
1. 尝试提取数学运算验证码文本（LoginPageHandler.extractCaptchaText）
   ├── 成功 → 自动计算结果（calculateCaptcha：支持 +、-、*、/ 运算）
   │         → 填入验证码输入框
   └── 失败 → 降级为图片验证码人工介入模式：
              a. 截取验证码图片保存到 .tmp/ 目录
              b. 轮询 .tmp/captcha_code.txt 等待人工写入
              c. 首次等待 300 秒，重试等待 120 秒
              d. 读取文件内容填入输入框
```

#### 4.7 勾选协议复选框

**负责组件**：`LoginPageHandler.checkAgreement()` / `BrowserSessionManager.checkPrivacyAgreement()`

通过 JS 查找所有未勾选的 `input[type="checkbox"]`，依次点击。

#### 4.8 点击登录按钮

**负责组件**：`LoginPageHandler.clickLoginButton()`

通过 JS 定位文本含 "登录" 或 class 含 "login" 的按钮，使用 `dispatchEvent(new MouseEvent('click'))` 点击，避免 Selenium 原生点击被拦截。

点击后等待 2 秒，检测页面变化：
- URL 不再含 `login.html` → 登录成功（无需滑块）
- 出现 `.yidun` 元素 → 进入滑块验证阶段

### 阶段四：网易云盾滑块验证（核心新增）

#### 4.9 检测网易云盾滑块

**负责组件**：`AutoLoginService.handleSliderCaptcha()`

**检测方式**：

```java
// 等待网易云盾容器出现（最多 10 秒）
WebDriverWait wait = new WebDriverWait(driver, Duration.ofSeconds(10));
WebElement yidunPanel = wait.until(
    ExpectedConditions.presenceOfElementLocated(
        By.cssSelector(".yidun_panel, .yidun, [class*='yidun']")

```

**检测失败处理**：如果 10 秒内未出现滑块，检查是否已直接登录成功（URL 变化）。

#### 4.10 提取背景图和拼图图片

**负责组件**：`AutoLoginService.extractSliderCaptchaData()`（需要新增/修改）

网易云盾的图片通过 `<img>` 标签的 `src` 属性或 CSS `background-image` 提供：

```javascript
// JS 提取图片 URL
(function() {
    var result = {};

    // 方式1：从 img 标签获取
    var bgImg = document.querySelector('.yidun_bg-img img, .yidun_bgimg img');
    var jigsawImg = document.querySelector('.yidun_jigsaw img, .yidun_slider__icon img');

    if (bgImg) result.bgUrl = bgImg.src;
    if (jigsawImg) result.sliderUrl = jigsawImg.src;

    // 方式2：从 CSS background-image 获取
    if (!result.bgUrl) {
        var bgDiv = document.querySelector('.yidun_bg-img, .yidun_bgimg');
        if (bgDiv) {
            var bg = getComputedStyle(bgDiv).backgroundImage;
            if (bg && bg !== 'none') {
                result.bgUrl = bg.replace(/^url\(["']?/, '').replace(/["']?\)$/, '');
            }
        }
    }
    if (!result.sliderUrl) {
        var sliderDiv = document.querySelector('.yidun_jigsaw, .yidun_slider__icon');
        if (sliderDiv) {
            var bg = getComputedStyle(sliderDiv).backgroundImage;
            if (bg && bg !== 'none') {
                result.sliderUrl = bg.replace(/^url\(["']?/, '').replace(/["']?\)$/, '');
            }
        }
    }

    // 获取渲染尺寸用于距离修正
    var bgElement = document.querySelector('.yidun_bg-img, .yidun_bgimg');
    if (bgElement) {
        result.renderWidth = bgElement.offsetWidth;
        result.renderHeight = bgElement.offsetHeight;
    }

    return JSON.stringify(result);
})()
```

**图片下载**：通过 OkHttp 下载图片到内存 `byte[]`，传给 `CaptchaService.calculateDistance()`。

#### 4.11 计算滑块距离

**负责组件**：`CaptchaService.calculateDistance()` + `calculateDistanceByEdge()`（已实现）

**算法流程**：

```
1. 主算法：灰度模板匹配（templateMatch）
   a. bgImage、sliderImage 分别转灰度矩阵
   b. 遍历背景图所有可能的 X 位置
   c. 在每个位置计算与拼图的像素差异总和（SAD）
   d. 差异最小的 X 即为缺口位置

2. 备用算法：边缘检测（calculateDistanceByEdge）
   a. 分析背景图的垂直边缘变化
   b. 找到边缘变化最大的 X 位置

3. 距离修正（新增）
   int correctedDistance = (int)(rawDistance * renderWidth / imageNativeWidth);
```

#### 4.12 生成滑动轨迹

**负责组件**：`CaptchaService.generateEaseSlideTrack()`（已实现）

**S 曲线缓动函数**：

```java
private double calculateEaseProgress(double progress) {
    if (progress < 0.3) {
        return 2 * progress * progress;              // 加速段
    } else if (progress < 0.7) {
        return -1 + (4 - 2 * progress) * progress;   // 匀速段
    } else {
        return 1 - Math.pow(-2 * progress + 2, 2) / 2; // 减速段
    }
}
```

**轨迹特征**：
- 轨迹点数：`30 + distance / 10`
- X 轴抖动：`±2px` 随机偏移
- Y 轴波动：`sin(i * 0.3) * 3` 正弦波
- 累积式位移：每步只移动增量 `deltaX = track[i].x - track[i-1].x`

#### 4.13 执行滑动操作

**负责组件**：需新增方法（基于现有 `CaptchaService` + `BrowserSessionManager`）

```java
public boolean executeYidunSlide(WebDriver driver, int distance) {
    // 1. 定位滑块手柄
    WebElement handler = driver.findElement(
        By.cssSelector(".yidun_slider__handler, .yidun_btn, [class*='yidun'] [class*='handler']")
    );

    // 2. 生成轨迹
    List<int[]> track = captchaService.generateEaseSlideTrack(distance);

    // 3. 执行拖动
    Actions actions = new Actions(driver);
    actions.clickAndHold(handler).perform();
    Thread.sleep(200); // 按住后短暂停顿

    int prevX = 0;
    for (int[] point : track) {
        int deltaX = point[0] - prevX;
        int deltaY = point[1] - track.get(0)[1]; // Y 相对于起点
        actions.moveByOffset(deltaX, deltaY).perform();
        prevX = point[0];
        Thread.sleep(15 + random.nextInt(10));
    }

    actions.release().perform();

    // 4. 等待验证结果
    Thread.sleep(2000);
    return checkYidunResult(driver);
}
```

#### 4.14 检测滑块验证结果

```java
private boolean checkYidunResult(WebDriver driver) {
    try {
        // 方式1：检查成功提示文本
        WebElement tips = driver.findElement(By.cssSelector(".yidun_tips__text"));
        String text = tips.getText();
        if (text.contains("成功") || text.contains("通过")) {
            return true;
        }

        // 方式2：检查滑块面板是否消失
        List<WebElement> panels = driver.findElements(By.cssSelector(".yidun_panel"));
        if (panels.isEmpty() || !panels.get(0).isDisplayed()) {
            return true;
        }

        // 方式3：检查 URL 是否已跳转（登录成功）
        return !driver.getCurrentUrl().contains("login.html");
    } catch (NoSuchElementException e) {
        // 元素不存在，可能已经跳转，视为成功
        return !driver.getCurrentUrl().contains("login.html");
    }
}
```

#### 4.15 滑块验证失败重试

最多重试 3 次。每次重试：
1. 等待网易云盾刷新图片（通常自动刷新）
2. 重新提取图片 URL
3. 重新计算距离
4. 重新生成轨迹（随机性保证每次不同）
5. 重新执行滑动

### 阶段五：登录结果处理

#### 4.16 检测登录成功

**负责组件**：`LoginPageHandler.isLoginSuccess()` + `AutoLoginService.isLoggedIn()`

```
判断条件（满足任一）：
```

#### 4.17 提取 Token

**负责组件**：`CookieManager.getToken()`

**多策略依次尝试**：

```
策略1：指定 Cookie 名称匹配
  → 遍历: token, Token, TOKEN, session_token, auth_token, access_token,
          jwt, tk, SESSION, sessionId, JSESSIONID, Authorization

策略2：Cookie 启发式匹配
  → 名称包含 token/session/sid/auth 且值长度 ≥ 12

策略3：localStorage 扫描
  → JS 遍历 localStorage 所有 key，匹配 token/accessToken/tk/Authorization 等

策略4：sessionStorage 扫描
  → 同上
```

#### 4.18 Token 同步

**负责组件**：`AutoLoginService.syncTokenFromBrowser()`

```
1. 轮询 CookieManager.getToken()，最多 8 次，每次间隔 600ms
2. 获取到 Token 后调用 ZXRequestUtils.setToken(token) 同步
3. ZXRequestUtils 内置 30 分钟 TTL 缓存
```

## 8. 错误处理

### 8.1 重试参数

```java
MAX_LOGIN_RETRY = 3;            // 完整登录流程重试
MAX_IMAGE_CAPTCHA_RETRY = 5;    // 图片验证码重试
MAX_SLIDER_RETRY = 3;           // 滑块验证重试
CAPTCHA_WAIT_FIRST_SEC = 300;   // 首次验证码等待（5 分钟）
CAPTCHA_WAIT_RETRY_SEC = 120;   // 重试验证码等待（2 分钟）
TOKEN_SYNC_MAX_RETRY = 8;       // Token 同步轮询次数
TOKEN_SYNC_INTERVAL_MS = 600;   // Token 同步轮询间隔
```

### 8.2 错误码

| 错误码 | 说明 | 处理方式 |
|--------|------|---------|
| E001 | 浏览器启动失败 | 检查 Chrome/WebDriver 版本兼容性 |
| E002 | iframe 切换失败 | 递归遍历所有 frame |
| E003 | 元素定位失败 | 多策略定位 + 增加等待时间 |
| E004 | 验证码提取/识别失败 | 降级为人工介入 |
| E005 | 滑块图片下载失败 | 重新提取 URL + 添加 Referer 头 |
| E006 | 滑块距离计算异常 | 切换备用算法（边缘检测） |
| E007 | 滑块验证被拒绝 | 重新生成轨迹 + 重试 |
| E008 | Token 获取失败 | 多策略依次尝试 + 等待重试 |
| E009 | 登录被拒绝 | 检查账号密码 / 等待冷却 |
