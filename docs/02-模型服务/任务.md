# 02-模型服务任务清单

> 模块：模型服务模块 (Model Service)
> 包路径：`com.stock.modelService`
> 版本：2.1.0

---

## 已完成任务

| # | 任务 | 说明 |
|---|------|------|
| 1 | 多模块架构搭建 | 独立 Maven 子模块，分层：api/service/model/domain/dto/config |
| 2 | LSTM 价格预测 | DJL 0.36.0 集成，模型权重存储于 MongoDB（binary），按 `mongo:ID` 加载 |
| 3 | FinBERT 情感分析 | 新闻文本情感分析，0-100 归一化分数，支持 Negative/Neutral/Positive 三分类 |
| 4 | 模型训练 REST API | 提供 HTTP 接口触发模型训练 |
| 5 | MongoDB 模型存储 | 模型权重二进制存入 MongoDB，无本地文件 I/O |
| 6 | Redis 预测结果缓存 | 预测结果缓存到 Redis 减少重复推理 |
