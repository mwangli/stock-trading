# 02-模型服务需求文档

## 文档信息

| 项目 | 内容 |
|------|------|
| 模块 | 模型服务模块 (Model Service) |
| 包路径 | com.stock.modelService |

## 1. 模块概述

AI 模型模块是系统的智能核心，负责股票价格预测和财经新闻情感分析。

## 4. 数据需求

### 4.1 输入数据

- LSTM: 60 日 K 线数据 (开高低收成交量)
- 情感分析：新闻标题 + 内容
- 微调闭环：交易执行结果（持仓成本、卖出价格）

### 4.2 输出数据

- LSTM: 涨的概率 (0-1), 预测价格变化率
- 情感分析：情感得分 (-1 到 1), 关联股票代码
- 评估报告：准确率、F1、夏普比率等指标
- 标注数据：文本、标签、来源

### 4.3 收益率计算规则

```
收益率 = (卖出价 - 买入价) / 买入价 × 100%

说明：
- 不扣除交易费用（佣金、印花税）
- 仅反映价格波动收益
- T日买入信号执行价格 vs T+1卖出执行价格
```

## 6. 评估指标阈值

| 指标 | 合格阈值 | 触发微调阈值 |
|------|----------|--------------|
| 准确率 | ≥ 70% | < 70% |
| F1 分数 | ≥ 60% | < 60% |
| 方向准确率 | ≥ 65% | < 65% |
| ROC-AUC | ≥ 0.70 | < 0.70 |
| 夏普比率 | ≥ 0.5 | < 0.5 |

## 7. 微调触发条件

满足以下任一条件时触发微调：
- 准确率 < 70%
- F1 < 60%
- 方向准确率 < 65%
- 连续 3 个评估周期准确率下降
- 夏普比率 < 0.5

触发微调还需满足：
- 标注样本数 ≥ 500 条
- 模型版本已稳定 7 天以上

## 8. 自动标注规则

| T+1 收益率 | 标注标签 |
|------------|----------|
| > 1% | 利好 (Positive) |
| -1% ~ 1% | 中性 (Neutral) |
| < -1% | 利空 (Negative) |
