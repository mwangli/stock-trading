## 1. 模块概述

策略模块是系统的决策核心，包含两类策略：
1. **选股策略**：基于 AI 模型预测结果生成股票评分和买入信号 (天级)
2. **T+1 卖出策略**：基于分钟级价格走势的实时卖出决策 (分钟级)

| 模式名称 | 移动止损 | RSI | 成交量 | 布林带 | 卖出阈值 | 适用场景 |

> 说明：表格格式为 "权重/阈值"

#### 2.6.4 自动切换规则

**市场状态触发：**
```
触发条件                          切换动作
────────────────────────────────────────────
大盘跌幅 > 5%                   → 切换到保守模式
大盘跌幅 > 3%                   → 提高卖出阈值 10%
大盘涨幅 > 3%                   → 切换到激进模式
VIX 波动率指数 > 25             → 切换到保守模式
```

**策略熔断规则：**
```
熔断条件                          切换动作
────────────────────────────────────────────
单个指标连续失败 N 次（默认3次）  → 禁用该指标
单个指标当日失败超过 M 次（默认10次） → 禁用该指标

策略熔断后恢复时间（默认1小时）   → 自动恢复原配置
```

**时段切换规则：**
```
时段                          策略配置
────────────────────────────────────────────
开盘 30 分钟 (9:30-10:00)     → 激进模式（捕捉开盘行情）

#### 2.7.2 学习优化算法

**优化目标：**
```
主要优化目标（按优先级）：
1. 最大化收益 - 总收益率
2. 高点捕获率 - 卖出价/当日最高价 > 75%
3. 胜率 - 盈利交易/总交易 > 60%
4. 最大回撤控制 - < 15%
```

**优化方法：**
```
1. 网格搜索 (Grid Search)
   - 适用场景：参数空间较小
   - 优点：简单可靠，可找到全局最优
   - 缺点：计算量大

2. 遗传算法 (Genetic Algorithm)
   - 适用场景：参数空间大，多目标优化
   - 优点：效率高，可处理复杂问题
   - 缺点：可能陷入局部最优

3. 贝叶斯优化 (Bayesian Optimization)
   - 适用场景：计算成本高的优化
   - 优点：样本效率高
   - 缺点：实现复杂
```

**可优化参数范围：**
```
参数类型          参数名              默认值    优化范围
─────────────────────────────────────────────────────
移动止损          权重                40%       20%-50%
移动止损          回撤容忍度          2%        1%-5%
RSI 超买          权重                20%       10%-40%
RSI 超买          超买阈值           75        70-85
成交量背离        权重                20%       10%-40%
成交量背离        萎缩阈值           0.8       0.6-0.9
布林带            权重                20%       10%-40%
布林带            突破阈值           1.005     1.003-1.010
动态阈值          高收益阈值         50%       40%-60%
动态阈值          正常阈值           60%       50%-70%
动态阈值          低收益阈值         70%       60%-80%
```

## 3. 算法需求

### 3.1 综合得分计算 (选股策略)

```
综合得分 = LSTM 因子得分 × LSTM权重 + 情感因子得分 × 情感权重
默认权重：LSTM=0.6, 情感=0.4

# 因子标准化算法
LSTM 因子得分 = 涨的概率
# 说明：LSTM模型输出的涨跌概率本身就是 [0, 1] 区间，无需额外转换
# 概率 > 0.5 表示看涨，概率 < 0.5 表示看跌

情感因子得分 = (情感得分 + 1) / 2
# 说明：情感得分范围 [-1, 1]，映射到 [0, 1]
# 情感得分 > 0 表示正面，情感得分 < 0 表示负面

# 得分解释
综合得分范围：[0, 1]
- 得分 ≥ 0.7：强烈推荐买入
- 得分 0.5~0.7：推荐买入
- 得分 0.3~0.5：中性观望
- 得分 < 0.3：不推荐
```

### 3.2 股票排名

- 按综合得分降序排序
- 输出所有股票的排名和得分

### 3.3 信号生成规则

```
========================================
买入信号规则
========================================

触发条件（必须同时满足）：
1. 综合排名前 N（默认 N=3）
2. 当前无持仓（已持仓股票不重复买入，顺延到下一名）
3. 综合得分 ≥ 最低得分阈值（默认 0.4）

信号强度计算：
- 第 1 名：强度 = 100
- 第 2 名：强度 = 85
- 第 3 名：强度 = 70
- 第 N 名：强度 = 100 - (N-1) × 15

买入数量计算：
- 单只股票买入金额 = 可用资金 × 单只权重 / 推荐数量
- 买入股数 = 买入金额 / 当前价格（向下取整到100股）

========================================
卖出信号规则（持仓股排名）
========================================

触发条件：
1. 持仓股综合排名跌出前 M（默认 M=10）
2. 持仓时间 ≥ 1 个交易日（T+1 规则）

信号强度计算：
- 信号强度 = 50 + (M - 当前排名) × 5
- 排名越靠后，卖出信号越强
- 示例：M=10, 排名15时，强度 = 50 + (10-15) × 5 = 25（弱卖出）
- 示例：M=10, 排名20时，强度 = 50 + (10-20) × 5 = 0（强卖出）

========================================
T+1 卖出信号规则（分钟级）
========================================

卖出决策优先级（从高到低）：

优先级 1 - 强制卖出（无条件立即执行）：
├── 尾盘强制卖出 (14:57) → 立即市价卖出
└── 移动止损触发 → 立即卖出（保护本金，优先级最高）

优先级 2 - 综合得分判断（条件执行）：
├── 计算综合得分 = Σ(指标触发状态 × 指标权重)
├── 获取动态阈值（根据当日涨幅）
└── 综合得分 ≥ 动态阈值 → 执行卖出

注意：优先级1的信号无需计算综合得分，直接执行
```

### 3.4 T+1 卖出策略算法

**移动止损计算:**
```
if 当前价 > 最高价:
    最高价 = 当前价
    止损线 = 最高价 × (1 - 回撤容忍度)

回撤容忍度:
- 基础值：2%
- 高波动股票 (日振幅>5%)：3%
- 低波动股票 (日振幅<3%)：1%

触发条件：当前价 <= 止损线
```

**RSI 超买计算:**
```
RS = 平均上涨幅度 (14) / 平均下跌幅度 (14)
RSI = 100 - [100 / (1 + RS)]

触发条件：RSI(14) > 75 AND 当前价 > 开盘价 × 1.02
```

**成交量背离计算:**
```
当前价是最近 5 分钟最高 AND
最近 1 分钟成交量 < 前 5 分钟均量 × 0.8

触发条件：量价背离确认
```

**布林带突破计算:**
```
中轨 = MA(close, 20)
上轨 = 中轨 + 2 × STD(close, 20)

触发条件：
当前价 > 上轨 × 1.005 AND 连续 3 分钟在上轨之上
```

**综合得分计算 (T+1 策略):**
```
指标           权重    触发时得分
移动止损       40%     40
RSI 超买        20%     20
成交量背离     20%     20
布林带上轨     20%     20
总分           100%    100

总分 = Σ(指标触发状态 × 指标权重)
```

```
当日涨幅          卖出阈值    逻辑说明
────────────────────────────────────────────────
> 3%             50%        已有不错收益，可更积极锁定利润
1% ~ 3%          60%        正常情况，标准卖出条件
0% ~ 1%          70%        收益有限，需更强卖出信号
< 0%             80%        亏损状态，需非常强的信号才卖出

说明：动态阈值仅用于综合得分判断，不影响强制卖出信号（移动止损、尾盘强制）
```
