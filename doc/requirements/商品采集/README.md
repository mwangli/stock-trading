## 功能描述

商品采集模块是AI Shopping系统的核心数据采集层，负责从第三方电商平台（Takealot.com）自动采集商品数据，并进行持久化存储。

### 核心功能

1. **分类信息采集**
   - 自动获取电商平台所有商品分类
   - 支持分类层级管理（1-3级）
   - 分类数据持久化存储

2. **商品数据采集**
   - 按分类批量采集商品
   - 支持TOP N商品采集
   - 完整商品信息抓取（16个字段）

3. **品牌过滤系统**
   - 知名品牌黑名单维护
   - 自动品牌识别和过滤
   - 规避侵权风险

4. **数据持久化**
   - 商品数据写入数据库
   - 重复数据检测和更新
   - 采集历史记录

5. **定时自动采集**
   - Spring Scheduler定时任务
   - 支持Cron表达式配置
   - 任务执行状态监控

6. **API接口服务**
   - 商品列表查询接口
   - 商品详情查询接口
   - 分类列表查询接口
   - 采集任务管理接口
