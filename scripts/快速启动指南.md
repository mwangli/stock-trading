# 快速启动指南 - 哈工大中文 BERT 情感分析

## 概述

本指南将帮助你在 **15 分钟内** 完成中文情感分析模型的下载、配置和测试。

---

## 前置要求

- ✅ Git 已安装
- ✅ Git LFS 已安装（下载大文件必需）
- ✅ Java 17+
- ✅ 网络连接正常

### 安装 Git LFS（如果未安装）

**Windows**:
```bash
# 1. 下载并安装：https://git-lfs.com
# 2. 验证安装
git lfs --version
```

**检查是否已安装**：
```bash
git lfs --version
```

如果显示版本号，说明已安装。

## 快速启动步骤

### 步骤 1：下载模型（10 分钟）

**方式 A：使用脚本（推荐）**

```bash
cd D:\ai-stock-trading
scripts\download-bert-model.bat
```

**方式 B：手动下载**

```bash
cd D:\ai-stock-trading\models\sentiment-analysis

# 初始化 Git LFS
git lfs install

# 克隆模型仓库
git clone https://huggingface.co/hfl/chinese-bert-wwm-ext

# 移动文件
cd chinese-bert-wwm-ext
move * ..\
cd ..
rmdir /s /q chinese-bert-wwm-ext
```

**验证下载**：
```bash
cd D:\ai-stock-trading\models\sentiment-analysis
dir

# 应该看到以下文件：
# - pytorch_model.bin  (约 400MB)
# - config.json        (约 1KB)
# - vocab.txt          (约 200KB)
```

配置已自动设置，无需修改。

## 常见问题

### Q1: Git LFS 下载失败

**错误信息**：
```
git-lfs: command not found
```

**解决方案**：
1. 下载 Git LFS：https://git-lfs.com
2. 安装后重启终端
3. 运行：`git lfs install`

---

### Q2: 模型下载太慢

**解决方案 1**：使用镜像
```bash
git clone https://hf-mirror.com/hfl/chinese-bert-wwm-ext
```

**解决方案 2**：手动下载
1. 访问：https://huggingface.co/hfl/chinese-bert-wwm-ext
2. 点击 "Files and versions"
3. 下载：`pytorch_model.bin`, `config.json`, `vocab.txt`
4. 放到 `models/sentiment-analysis/` 目录

---

### Q3: 模型加载失败

**错误信息**：
```
模型文件不存在
```

**检查清单**：
```bash
cd models\sentiment-analysis
dir

# 必需文件：
# ✓ pytorch_model.bin (400MB+)
# ✓ config.json
# ✓ vocab.txt
```

**解决方案**：重新下载模型

---

### Q4: 端口被占用

**错误信息**：
```
Port 8080 is already in use
```

**解决方案**：
```bash
# 方法 1: 修改端口
# 编辑 application.yml，修改 server.port: 8081

# 方法 2: 关闭占用进程
netstat -ano | findstr :8080
taskkill /PID <PID> /F
```
