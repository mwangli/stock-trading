# 03-策略模块 - 设计文档

## 模块概述

策略模块（`com.stock.strategy`）是系统的决策核心层，基于 AI 模型的预测结果和情感分析，生成股票评分和交易信号。

## 核心职责

1. **综合选股** - 基于 LSTM 预测 (60%) 和情感分析 (40%) 双因子选股
2. **股票评分** - 计算每只股票的综合得分并排名
3. **决策引擎** - 根据评分和策略规则生成交易信号
4. **信号管理** - 交易信号的生成、存储和分发
5. **策略配置** - 支持策略参数调整和热更新

## 核心组件

### 1. StockSelector (选股器)

- **职责**: 执行综合选股策略
- **输入**: 所有股票的 AI 预测结果和情感结果
- **输出**: 推荐股票列表 (Top N)
- **策略**: 双因子加权评分

### 2. FactorCalculator (因子计算器)

- **职责**: 计算各因子得分
- **因子**:
  - LSTM 因子：涨跌概率归一化
  - 情感因子：情感得分归一化
- **标准化**: Min-Max 归一化到 [0,1] 区间

### 3. RankingService (排名服务)

- **职责**: 计算综合得分并排序
- **公式**: `综合得分 = LSTM 得分 × 0.6 + 情感得分 × 0.4`
- **输出**: 按得分降序排列的股票列表

### 4. DecisionEngine (决策引擎)

- **职责**: 根据选股结果生成交易信号
- **输入**: 股票排名、当前持仓、市场状态
- **输出**: 买入/卖出信号
- **规则**:
  - Top 1: 强烈推荐买入
  - Top 2-3: 推荐买入 (备选)
  - 持仓股跌出前 10: 考虑卖出

### 5. SignalGenerator (信号生成器)

- **职责**: 生成标准化的交易信号
- **信号类型**:
  - STRONG_BUY (强烈买入)
  - BUY (买入)
  - HOLD (持有)
  - SELL (卖出)
  - STRONG_SELL (强烈卖出)
- **信号强度**: 0-100

## 核心算法

### 双因子选股算法

```
综合得分 = LSTM 因子得分 × 0.6 + 情感因子得分 × 0.4

其中:
- LSTM 因子得分 = (涨的概率 - 0.5) × 2  [归一化到 0-1]
- 情感因子得分 = (情感得分 + 1) / 2  [从 -1~1 映射到 0-1]
```

### 股票排名算法

```
1. 计算每只股票的综合得分
2. 按综合得分降序排序
3. 选出 Top N 股票 (默认 N=10)
4. 标记推荐等级:
   - 第 1 名：强烈推荐 (5 星)
   - 第 2-3 名：推荐 (4 星)
   - 第 4-6 名：关注 (3 星)
   - 第 7-10 名：观察 (2 星)
```

### 交易信号生成规则

```
买入信号:
- 综合排名前 3 且无持仓 → 买入信号
- 信号强度 = 排名分数 (第 1 名=100, 第 2 名=80, 第 3 名=60)

卖出信号:
- 持仓股跌出前 10 → 卖出信号
- 持仓股情感转为利空 → 卖出信号
- 信号强度 = 50 + (10 - 排名) × 5
```

## 数据流程

### 综合选股流程

```
AI 预测结果 + 情感结果
    ↓
因子计算器 (标准化)
    ↓
综合得分计算 (加权求和)
    ↓
排名排序 (降序)
    ↓
Top N 筛选 (默认 10)
    ↓
输出：推荐股票列表
```

### 交易信号生成流程

```
股票排名 + 当前持仓
    ↓
决策引擎 (规则匹配)
    ↓
信号生成器 (标准化)
    ↓
信号存储
    ↓
输出：交易信号列表
```

## 策略配置

### 可配置参数

| 参数名 | 默认值 | 说明 |
|--------|--------|------|
| lstmWeight | 0.6 | LSTM 因子权重 |
| sentimentWeight | 0.4 | 情感因子权重 |
| topN | 10 | 选股数量 |
| sellThreshold | 10 | 卖出阈值 (排名低于此值考虑卖出) |

### 权重调整策略

**保守型:**
- LSTM 权重：0.7
- 情感权重：0.3
- 特点：更相信模型预测，减少情绪波动影响

**激进型:**
- LSTM 权重：0.5
- 情感权重：0.5
- 特点：平衡模型和新闻情绪

**模型优先:**
- LSTM 权重：0.8
- 情感权重：0.2
- 特点：高度依赖 LSTM 预测

## 关键设计决策

### 1. 双因子模型

- **选择**: LSTM(60%) + 情感 (40%)
- **原因**:
  - LSTM 捕捉价格趋势规律
  - 情感捕捉市场情绪和突发事件
  - 两者互补，提高预测准确性

### 2. 权重设计

- **默认**: LSTM 权重更高 (60%)
- **原因**:
  - 技术指标相对可靠
  - 新闻情绪波动大，噪声多
  - 但不可完全忽略，保留 40%

### 3. 排名优于绝对值

- **设计**: 使用相对排名而非绝对得分
- **原因**:
  - 排名更稳定，不受市场整体波动影响
  - 便于横向比较所有股票
  - 符合人类决策习惯

### 4. 信号分级

- **设计**: 5 级信号强度
- **优势**:
  - 执行模块可以根据强度调整仓位
  - 避免非黑即白的简单决策
  - 支持更精细的风控策略

## 策略评估

### 评估指标

1. **准确率**: 推荐股票实际上涨的比例
2. **收益率**: 跟随推荐的平均收益率
3. **夏普比率**: 风险调整后收益
4. **最大回撤**: 最大亏损幅度

### 评估周期

- **日评估**: 每日推荐准确率
- **周评估**: 周收益率统计
- **月评估**: 月度综合表现
- **季评估**: 季度策略调整
