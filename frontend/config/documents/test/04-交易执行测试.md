## 3. 系统测试

### 3.1 端到端测试

**测试场景：完整买入流程**
```
1. 接收交易信号 (买入)
2. 风控检查 (止损/仓位/T+1)
3. 生成买入订单
4. 执行交易 (模拟)
5. 更新持仓 (增加)
6. 更新账户 (扣减资金)
7. 记录交易历史
```

**测试场景：完整卖出流程**
```
1. 接收交易信号 (卖出) 或 止损触发
2. 风控检查 (T+1/跌停)
3. 生成卖出订单
4. 执行交易 (模拟)
5. 更新持仓 (减少)
6. 更新账户 (增加资金)
7. 计算盈亏
8. 记录交易历史
```

**测试场景：风控拒绝流程**
```
1. 接收交易信号
2. 风控检查失败
3. 拒绝交易
4. 记录拒绝原因
5. 不执行交易
```

## 6. 风控规则测试

### 6.1 止损规则测试

| 测试场景 | 预期结果 |
|----------|----------|
| 日亏损 2.9% | 允许买入 |
| 日亏损 3.1% | 拒绝买入 |
| 月亏损 9.9% | 允许买入 |
| 月亏损 10.1% | 拒绝买入 |
| 个股亏损 7.9% | 不触发卖出 |
| 个股亏损 8.1% | 触发卖出 |

### 6.2 仓位规则测试

| 测试场景 | 预期结果 |
|----------|----------|
| 单股仓位 29% | 允许买入 |
| 单股仓位 31% | 拒绝买入 |
| 总仓位 79% | 允许买入 |
| 总仓位 81% | 拒绝买入 |

### 6.3 T+1 规则测试

| 测试场景 | 预期结果 |
|----------|----------|
| 当日买入，当日卖出 | 拒绝 |
| 当日买入，次日卖出 | 允许 |
