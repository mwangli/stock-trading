# 情感分析模型微调迭代闭环方案

## Why

当前情感分析模型仅加载预训练模型做推理，未实现真正的微调训练。当模型预测准确率和收益率未达预期时，需要一套自动化、可量化的模型迭代优化机制。

## What Changes

- **新增**：模型准确率定量评估体系（评估指标、阈值标准、触发机制）
- **新增**：收益率定量验证方案（简化计算：仅基于买卖价差）
- **新增**：自动标注管道（基于交易反馈，无需人工参与）
- **新增**：模型微调闭环集成（评估→标注→微调→上线）
- **修改**：收益率计算方法（去除交易费用，仅反映价格波动收益）

## Impact

- **受影响的规格**：
  - `MODELS-REQ-013` 增量迭代训练（扩展为完整闭环）
  - `MODELS-REQ-015` 增量训练效果评估（新增评估指标体系）
- **受影响的代码**：
  - `SentimentTrainerService` - 扩展评估与标注功能
  - `SentimentController` - 新增评估触发接口
  - `JobSchedulerService` - 新增微调调度任务类型
  - MongoDB Collections - 新增评估结果、标注数据、模型版本存储

## ADDED Requirements

### Requirement: 模型准确率定量评估

系统 SHALL 提供情感模型准确率的定量评估能力，包括：
- 准确率、F1分数、召回率、ROC-AUC 等核心指标计算
- 标准化测试数据集划分（训练70%/验证15%/测试15%，时序划分）
- 准确率阈值标准与迭代触发机制

#### Scenario: 评估触发
- **WHEN** 每日收盘后或手动触发评估
- **THEN** 系统计算各项指标，与阈值对比，输出评估报告

#### Scenario: 触发微调
- **WHEN** 任一指标低于触发阈值或连续3周期下降
- **THEN** 系统自动触发微调训练流程

### Requirement: 收益率定量验证

系统 SHALL 提供基于交易反馈的收益率量化验证能力：
- 收益率 = (卖出价 - 买入价) / 买入价 × 100%（不含交易费用）
- 统计周期：日/周/月/累计
- 风险调整指标：夏普比率、最大回撤

#### Scenario: 收益率计算
- **WHEN** T+1卖出执行后
- **THEN** 系统计算收益率并记录，用于后续标注

### Requirement: 自动标注管道

系统 SHALL 提供基于交易反馈的自动标注能力，无需人工参与：
- T日收盘信号预测 → T+1实际结果 → 标注标签生成
- 标注规则：收益>阈值 → 利好；收益<负阈值 → 利空；其他 → 中性
- 标注数据质量审核机制

#### Scenario: 自动标注
- **WHEN** T+1交易完成并计算收益率后
- **THEN** 系统自动生成标注数据并存储

### Requirement: 模型微调闭环

系统 SHALL 提供完整的模型微调闭环：
- 微调触发条件可配置（准确率阈值、夏普比率阈值等）
- 支持模型版本管理（版本记录、回滚能力）
- 微调完成后自动验证效果，不达标则回滚

#### Scenario: 微调执行
- **WHEN** 评估结果触发微调条件且标注样本充足
- **THEN** 系统执行微调训练，验证效果，上线新模型或回滚

### Requirement: 模型版本管理

系统 SHALL 提供情感模型版本管理能力：
- 记录每次微调的版本、参数、评估指标
- 支持版本对比和快速回滚
- 与 JobSchedulerService 深度集成

## MODIFIED Requirements

### Requirement: MODELS-REQ-013 增量迭代训练

**原描述**：基于预训练模型微调，避免从头训练

**修改为**：实现完整的模型微调迭代闭环，包括评估→自动标注→微调训练→版本管理的全流程自动化

### Requirement: MODELS-REQ-015 增量训练效果评估

**原描述**：提供知识保留率、新数据准确率指标

**修改为**：提供完整的评估指标体系，包括准确率≥70%、F1≥60%、方向准确率≥65%、夏普比率≥0.5，并支持触发机制

## Key Parameters

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `evaluation.threshold.accuracy` | 70% | 准确率合格阈值 |
| `evaluation.threshold.f1` | 60% | F1合格阈值 |
| `evaluation.threshold.directionAccuracy` | 65% | 方向准确率合格阈值 |
| `evaluation.threshold.sharpeRatio` | 0.5 | 夏普比率合格阈值 |
| `evaluation.trigger.consecutiveDecline` | 3 | 连续下降触发周期数 |
| `fineTune.minSamples` | 500 | 最少标注样本数触发微调 |
| `fineTune.rollbackThreshold` | 5% | 回滚准确率降幅阈值 |
